📚 Sistema de Perseguição Multi‑Piso – Documentação Técnica
==========================================================
Projeto MCR – Versão 1.0
Junho de 2026

🎯 Visão Geral
O sistema permite que qualquer criatura (summons, pets, monstros agressivos) siga o seu mestre ou alvo através de diferentes pisos (eixo Z), utilizando escadas, buracos, teleports, ladders e itens interativos do mapa. A perseguição é 100% nativa, reutilizando o sistema de walk e pathfinding do Canary.

Princípios fundamentais
Fila de rastros no jogador: cada mudança de piso do jogador é registada numa std::deque<Pair<Position, Position>> (origem, destino).

Máquina de estados no Monster: idle (0) → rota (1) → retorno ao spawn (2).

Movimento 100% nativo: startAutoWalk + FLAG_NOLIMIT para o último passo.

Fallback para itens interativos: se o tile de origem não transportar automaticamente (bueiros, escadas de corda), teleporta diretamente para o destino após 2 ciclos.

Fidelidade do summon: a fila só é limpa se o jogador estiver acessível (pathfinding). Se estiver visível mas inacessível, o summon continua a seguir a fila.

bool Creature::goToPosition(const Position& target) {
    if (isRemoved() || isDead()) return false;

    std::vector<Direction> listDir;
    FindPathParams fpp;
    getPathSearchParams(getCreature(), fpp);
    fpp.minTargetDist = 0;
    fpp.maxTargetDist = 1;
    fpp.fullPathSearch = true;
    fpp.clearSight = false;
    fpp.maxSearchDist = 0;

    if (!getPathTo(target, listDir, fpp) || listDir.empty()) return false;

    startAutoWalk(listDir);
    setWalkingToPosition(true);
    return true;
}
Função: Calcula um caminho até uma posição fixa e inicia o walk nativo. O flag m_walkingToPosition impede que o goToFollowCreature interfira.

Creature::onCreatureDisappear
Modificado para não remover o alvo se a criatura apenas mudou de piso (z diferente e criatura ainda existe).

3.2 monster.cpp – Monster::followTargetAcrossFloor (versão final)
Estado 0 – Avaliação da fila
Itera a m_floorChangeQueue do jogador (até 3 transições para hostis, todas para summons).

Para cada transição com org.z == myPos.z, verifica se o pathfinding consegue chegar a org.

Se encontrada, define m_origin, m_dest, muda para estado 1 e remove a transição da fila.

Se nenhuma for válida, summons teleportam para perto do mestre; hostis desistem após 3 falhas (estado 2).

Estado 1 – A caminho da origem
Se distance > 1: recalcula o caminho com getPathTo(m_origin, ..., maxTargetDist=1) e chama startAutoWalk.

Se distance == 1: força o último passo com g_game().internalMoveCreature(dir, FLAG_NOLIMIT). Esta flag ignora restrições de queryAdd, permitindo pisar o tile da escada. O motor então chama queryDestination e transporta a criatura.

Se distance == 0: o monstro está no waypoint.

Se o tile for uma escada normal, o motor transporta automaticamente.

Se após 2 ciclos (m_stuckCount >= 2) o monstro ainda estiver no mesmo piso, assume que é um item interativo (bueiro, escada de corda) e teleporta diretamente para m_dest.

Timeout: 15 segundos para hostis, 120 segundos para summons.

Estado 2 – Regresso ao spawn (apenas hostis)
Usa goToPosition(masterPos) para caminhar de volta ao spawn com movimento nativo.

Se o pathfinding falhar, teleporta diretamente.

Timeout de 30 segundos.

Abandono da rota
Se o jogador estiver no mesmo piso, a fila estiver vazia, a distância ≤ 20 tiles e o pathfinding confirmar acessibilidade, a rota é abandonada e o followCreature original é restaurado.

Se o jogador estiver visível mas inacessível (ex.: atrás de uma parede), o summon mantém a rota, seguindo a fila de rastros.

3.3 monster.cpp – Monster::onWalkComplete
cpp
void Monster::onWalkComplete() {
    if (m_state == 1) return; // não limpa durante perseguição
    if (isWalkingToPosition()) {
        setWalkingToPosition(false);
        setFollowCreature(m_savedFollowCreature.lock());
        return;
    }
    if (isSummon() && getMaster()) {
        setFollowCreature(getMaster());
    }
}
Função: Durante o estado 1, o onWalkComplete é ignorado para não interferir com a sequência de passos. O transporte de piso é detetado pelo onCreatureMove ou pelo fallback de itens interativos.

🧪 Comportamento Final
Situação	Summon/Pet	Monstro Agressivo
Jogador muda de piso (escada normal)	Caminha nativamente até à escada, pisa com FLAG_NOLIMIT, é transportado.	Igual, mas vê apenas as 3 últimas transições.
Jogador muda de piso (item interativo)	Pisa o tile, após 2 ciclos teleporta para o destino.	Igual.
Jogador no mesmo piso e acessível	Abandona a rota, restaura follow normal.	Igual.
Jogador visível mas inacessível	Mantém a rota, segue a fila de rastros.	Igual.
Timeout (15s hostil / 120s summon)	Teleporta para perto do mestre.	Entra em estado 2 (regresso ao spawn).
Regresso ao spawn	N/A (não entra em estado 2).	Caminha nativamente até ao spawn; teleporta se o pathfinding falhar.
⚠️ Limitações Conhecidas
O movimento para o spawn (estado 2) pode falhar em mapas complexos, ativando o teleporte de fallback.

Monstros hostis só veem 3 transições; se o jogador usar mais de 3 escadas rapidamente, o monstro perde o rasto.

Itens interativos requerem 2 ciclos (≈2 segundos) para o fallback de teleporte, o que pode parecer uma pequena pausa.

Com esta documentação, qualquer desenvolvedor pode compreender, manter e expandir o sistema de perseguição multi‑piso. A arquitetura de fila e estados está preparada para ser reutilizada noutros contextos (NPCs, patrulhas, etc.).


void Monster::followTargetAcrossFloor() {
    std::shared_ptr<Creature> target;
    if (!m_attackedCreature.expired()) target = m_attackedCreature.lock();
    if (!target) target = getFollowCreature();
    if (!target && isSummon()) target = getMaster();

    bool isHostile = !isSummon();
    Position myPos = getPosition();

    auto targetPlayer = target ? target->getPlayer() : nullptr;
    if (!targetPlayer && isSummon()) {
        auto master = getMaster();
        if (master) targetPlayer = master->getPlayer();
    }

    g_logger().info("[MCR-FLOOR] {}::followTargetAcrossFloor() estado={} pos={}", getName(), m_state, myPos.toString());

    // -----------------------------------------------------------
    // ESTADO 2: Regresso ao spawn (movimento nativo)
    // -----------------------------------------------------------
    if (m_state == 2 && isHostile) {
        g_logger().info("[MCR-FLOOR] {} estado=2 a regressar ao spawn. pos={} spawn={}", 
            getName(), myPos.toString(), masterPos.toString());

        if (myPos == masterPos) {
            g_logger().info("[MCR-FLOOR] {} chegou ao spawn. estado=0", getName());
            if (auto saved = m_savedFollowCreature.lock()) setFollowCreature(saved);
            m_state = 0; m_stuckCount = 0; m_cycleCount = 0;
            setWalkingToPosition(false);
            return;
        }

        if (!isWalkingToPosition()) {
            m_savedFollowCreature = getFollowCreature();
            setFollowCreature(nullptr);

            std::vector<Direction> listDir;
            FindPathParams fpp;
            getPathSearchParams(getMonster(), fpp);
            fpp.minTargetDist = 0;
            fpp.maxTargetDist = 0;
            fpp.fullPathSearch = true;
            fpp.clearSight = false;
            fpp.maxSearchDist = 0;

            if (getPathTo(masterPos, listDir, fpp) && !listDir.empty()) {
                startAutoWalk(listDir);
                setWalkingToPosition(true);
                g_logger().info("[MCR-FLOOR] {} estado=2 caminhada iniciada: {} passos", getName(), listDir.size());
            } else {
                g_logger().info("[MCR-FLOOR] {} estado=2 FALHA – teleportando", getName());
                setFollowCreature(m_savedFollowCreature.lock());
                g_game().internalTeleport(static_self_cast<Monster>(), masterPos);
                m_state = 0; m_stuckCount = 0; m_cycleCount = 0;
                setWalkingToPosition(false);
            }
            return;
        }

        m_cycleCount++;
        if (m_cycleCount > 30) {
            g_logger().info("[MCR-FLOOR] {} estado=2 TIMEOUT – teleportando", getName());
            setWalkingToPosition(false);
            setFollowCreature(m_savedFollowCreature.lock());
            g_game().internalTeleport(static_self_cast<Monster>(), masterPos);
            m_state = 0; m_stuckCount = 0; m_cycleCount = 0;
        }
        return;
    }

    if (!target) {
        g_logger().info("[MCR-FLOOR] {} sem alvo, a abortar.", getName());
        return;
    }

    // -----------------------------------------------------------
    // Se o jogador está no mesmo piso e ACESSÍVEL, ABANDONA a rota
    // -----------------------------------------------------------
    if (target->getPosition().z == myPos.z && targetPlayer && !targetPlayer->hasPendingFloorChange()) {
        int dist = std::max(std::abs(target->getPosition().x - myPos.x), std::abs(target->getPosition().y - myPos.y));
        if (dist <= 20) {
            bool reachable = false;
            {
                FindPathParams fpp;
                getPathSearchParams(getMonster(), fpp);
                std::vector<Direction> dummy;
                reachable = getPathTo(target->getPosition(), dummy, fpp);
            }
            if (reachable) {
                g_logger().info("[MCR-FLOOR] {} jogador acessivel no mesmo piso, a abandonar rota.", getName());
                if (auto saved = m_savedFollowCreature.lock()) setFollowCreature(saved);
                m_hasTransition = false;
                m_state = 0; m_stuckCount = 0; m_cycleCount = 0;
                m_pathList.clear(); m_noProgressCount = 0;
                m_walkingToWaypoint = false;
                setWalkingToPosition(false);
                return;
            }
        }
    }

    // -----------------------------------------------------------
    // ESTADO 1: A caminho da origem (movimento nativo)
    // -----------------------------------------------------------
    if (m_state == 1) {
        int dx = m_origin.x - myPos.x;
        int dy = m_origin.y - myPos.y;
        int distance = std::max(std::abs(dx), std::abs(dy));

        g_logger().info("[MCR-FLOOR] {} estado=1 origem={} destino={} dist={} walking={} listWalkDir={}", 
            getName(), m_origin.toString(), m_dest.toString(), distance, isWalkingToPosition() ? "sim" : "nao", listWalkDir.size());

        // Se está a caminhar e a listWalkDir ainda tem passos, aguarda.
        if (isWalkingToPosition() && !listWalkDir.empty()) {
            m_cycleCount++;
            int timeout = isHostile ? 15 : 120;
            if (m_cycleCount > timeout) {
                g_logger().info("[MCR-FLOOR] {} estado=1 TIMEOUT, a abortar.", getName());
                setWalkingToPosition(false);
                setFollowCreature(m_savedFollowCreature.lock());
                if (targetPlayer) targetPlayer->clearNextFloorChange();
                m_hasTransition = false; m_state = 0; m_cycleCount = 0;
                if (isHostile) { m_stuckCount++; if (m_stuckCount > 3) { setAttackedCreature(nullptr); setFollowCreature(nullptr); m_state = 2; } }
            }
            return;
        }

        // Se a caminhada terminou (listWalkDir vazia) ou ainda não iniciou
        setWalkingToPosition(false);

        // Se já está no tile de origem (distância 0)
        if (distance == 0) {
            // Verifica se o tile tem transição (escada normal). Se sim, o motor trata.
            // Se não, após 2 ciclos no mesmo tile, forçamos o teleporte para o destino.
            if (m_origin == myPos && m_origin.z != m_dest.z) {
                m_stuckCount++;
                if (m_stuckCount >= 2) {
                    // O tile não transportou automaticamente (item interativo). Força o teleporte.
                    g_logger().info("[MCR-FLOOR] {} tile interativo – forçando teleporte para destino.", getName());
                    g_game().internalTeleport(static_self_cast<Monster>(), m_dest);
                    if (targetPlayer) targetPlayer->clearNextFloorChange();
                    m_hasTransition = false; m_state = 0; m_stuckCount = 0; m_cycleCount = 0;
                }
            } else {
                m_stuckCount = 0;
            }
            return;
        }

        // Se está a 1 tile de distância, força o último passo com FLAG_NOLIMIT.
        if (distance == 1) {
            Direction dir = getDirectionTo(myPos, m_origin);
            if (dir != DIRECTION_NONE) {
                g_logger().info("[MCR-FLOOR] {} estado=1 forçando último passo com FLAG_NOLIMIT.", getName());
                ReturnValue ret = g_game().internalMoveCreature(static_self_cast<Monster>(), dir, FLAG_NOLIMIT);
                if (ret == RETURNVALUE_NOERROR) {
                    // Movido para o tile de origem. Aguarda o transporte (ou o fallback acima).
                    return;
                }
            }
            // Fallback: teleporta para o destino se o movimento falhar.
            g_logger().info("[MCR-FLOOR] {} estado=1 FALHA no último passo, teleportando para o destino.", getName());
            setFollowCreature(m_savedFollowCreature.lock());
            Position safeDest = findFreeTileNear(m_dest);
            g_game().internalTeleport(static_self_cast<Monster>(), safeDest);
            if (isHostile && targetPlayer) targetPlayer->clearNextFloorChange();
            m_hasTransition = false; m_state = 0; m_cycleCount = 0;
            return;
        }

        // Distância >1: recalcula o caminho com maxTargetDist=1.
        std::vector<Direction> listDir;
        FindPathParams fpp;
        getPathSearchParams(getMonster(), fpp);
        fpp.minTargetDist = 0;
        fpp.maxTargetDist = 1;
        fpp.fullPathSearch = true;
        fpp.clearSight = false;
        fpp.maxSearchDist = 0;

        if (getPathTo(m_origin, listDir, fpp) && !listDir.empty()) {
            startAutoWalk(listDir);
            setWalkingToPosition(true);
            g_logger().info("[MCR-FLOOR] {} estado=1 caminho calculado: {} passos (maxTargetDist=1)", getName(), listDir.size());
        } else {
            g_logger().info("[MCR-FLOOR] {} estado=1 FALHA ao calcular caminho, teleportando para o destino.", getName());
            setFollowCreature(m_savedFollowCreature.lock());
            Position safeDest = findFreeTileNear(m_dest);
            g_game().internalTeleport(static_self_cast<Monster>(), safeDest);
            if (isHostile && targetPlayer) targetPlayer->clearNextFloorChange();
            m_hasTransition = false; m_state = 0; m_cycleCount = 0;
        }
        return;
    }

    // -----------------------------------------------------------
    // ESTADO 0: Avaliação – obter a próxima transição da fila
    // -----------------------------------------------------------
    if (m_state == 0) {
        g_logger().info("[MCR-FLOOR] {} estado=0 a avaliar fila do jogador...", getName());
        if (targetPlayer && targetPlayer->hasPendingFloorChange()) {
            int maxTrans = isHostile ? 3 : 999;
            int count = 0;
            bool found = false;

            const auto& queue = targetPlayer->getFloorChangeQueue();
            size_t queueSize = queue.size();
            g_logger().info("[MCR-FLOOR] {} fila do jogador tem {} transicoes.", getName(), queueSize);

            for (auto it = queue.begin(); it != queue.end() && count < maxTrans && count < queueSize; ++it, ++count) {
                Position org = it->first;
                Position dst = it->second;

                g_logger().info("[MCR-FLOOR] {} a testar transicao {}: org={} dst={} (meu z={})", 
                    getName(), count, org.toString(), dst.toString(), myPos.z);

                if (org.z != myPos.z) continue;

                FindPathParams fpp;
                getPathSearchParams(getMonster(), fpp);
                fpp.fullPathSearch = true;
                fpp.clearSight = false;
                fpp.maxSearchDist = 0;
                fpp.minTargetDist = 0;
                fpp.maxTargetDist = 1;
                std::vector<Direction> path;
                
                bool pathFound = getPathTo(org, path, fpp);
                g_logger().info("[MCR-FLOOR] {} pathfinding para org: {} (fullSearch=true, clearSight=false)", 
                    getName(), pathFound ? "SUCESSO" : "FALHA");

                if (pathFound && !path.empty()) {
                    m_origin = org;
                    m_dest = dst;
                    m_hasTransition = true;
                    m_state = 1;
                    m_pathList.clear();
                    m_cycleCount = 0;
                    m_lastDistance = 999;
                    m_noProgressCount = 0;
                    m_walkingToWaypoint = false;
                    setWalkingToPosition(false);
                    m_stuckCount = 0;
                    if (isHostile) {
                        m_pursuitStartPos = myPos;
                        m_stuckCount = 0;
                    }
                    found = true;
                    targetPlayer->popFloorChangeQueueElements(count + 1);
                    g_logger().info("[MCR-FLOOR] {} TRANSICAO VALIDA! org={} dest={} estado=1", 
                        getName(), org.toString(), dst.toString());
                    break;
                }
            }

            if (!found) {
                g_logger().info("[MCR-FLOOR] {} nenhuma transicao acessivel encontrada.", getName());
                if (!isHostile) {
                    Position safeDest = findFreeTileNear(target->getPosition());
                    g_game().internalTeleport(static_self_cast<Monster>(), safeDest);
                    m_stuckCount = 0;
                } else {
                    m_stuckCount++;
                    if (m_stuckCount > 3) {
                        setAttackedCreature(nullptr);
                        setFollowCreature(nullptr);
                        m_state = 2;
                        m_pathList.clear();
                        m_stuckCount = 0;
                    }
                }
            }
        } else {
            g_logger().info("[MCR-FLOOR] {} fila do jogador VAZIA.", getName());
            if (!isHostile) {
                Position safeDest = findFreeTileNear(target->getPosition());
                g_game().internalTeleport(static_self_cast<Monster>(), safeDest);
                m_stuckCount = 0;
            } else {
                m_stuckCount++;
                if (m_stuckCount > 3) {
                    setAttackedCreature(nullptr);
                    setFollowCreature(nullptr);
                    m_state = 2;
                    m_pathList.clear();
                    m_stuckCount = 0;
                }
            }
        }
        return;
    }
