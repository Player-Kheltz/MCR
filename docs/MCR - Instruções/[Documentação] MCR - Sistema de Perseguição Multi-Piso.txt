CÓDIGOS

1. VISÃO GERAL
O sistema de perseguição multi‑piso permite que criaturas (summons do jogador e monstros hostis) persigam um alvo através de múltiplos andares.
A base do sistema é uma fila de transições de escada (StairTransition) que o jogador vai deixando ao mudar de piso. As criaturas consultam essa fila e caminham até ao ponto de origem da escada.

Novidade na v6.0:
Foi introduzido um Mapa Cognitivo Global (GlobalMonsterMap), um grafo partilhado que armazena tiles caminháveis e as transições de piso conhecidas. Quando o rasto direto do jogador (waypoints e fila) não é suficiente, o monstro recorre a um BFS multi‑piso sobre esse mapa para encontrar rotas alternativas ou atalhos.
O mapa é construído incrementalmente por todos os monstros, que adicionam tiles ao escanear a sua vizinhança.

1.1 Tipos de Criaturas e Comportamentos
Tipo	Perseguição	Timeout	Retorno	Uso de Migalhas/Mapa Cognitivo
Summon	Segue o mestre através de escadas	120 ciclos (~120s)	Teleporta para o mestre após timeout	Waypoints se inacessível; mapa cognitivo (se ativado)
Hostil	Persegue o jogador que está a atacar	45 segundos reais	Regressa a pé ao spawn ou à posição inicial	Waypoints quando bloqueado; mapa cognitivo como prioridade

3. FLUXO PRINCIPAL DE PERSEGUIÇÃO
A cada ciclo de pensamento (onThink_async), o monstro verifica se deve iniciar ou continuar uma perseguição.

cpp
// Verificar se deve começar a perseguir
if (isSummon() || (!isSummon() && (!m_attackedCreature.expired() ||
    (m_persistentTargetPlayer && !m_persistentTargetPlayer->getStairTransitions().empty())))) {
    checkStairPursuit();
}
Novo: Antes de entrar no bloco de perseguição, se m_usingGlobalPath == true, o monstro executa followGlobalPathStep() e cancela a rota global se o líder ficar acessível (ver secção 6).
Dentro do bloco if (m_pursuingStairs), há uma tentativa proactiva de usar o mapa cognitivo quando a posição atual já está mapeada e a distância à escada é > 2 tiles. Quando o monstro fica preso (stuckCount >= 8), a primeira ação é tentar o mapa cognitivo.

4. MECANISMOS DE TIMEOUT E DESISTÊNCIA
(Mantidos da v5.7)

4.1 Timeout Total (Hostis)
45 segundos reais (getCurrentTimeMs() - m_pursuitStartTime > 45000).
Se expirar:

Se usou escadas, inicia iniciarRetornoSpawn.

Caso contrário, regressa a pé à m_homePosition.

4.2 Timeout de Progresso (Hostis)
4 segundos sem reduzir a distância ao waypoint → abandona a escada atual e avança m_nextStairIndex.

4.3 Falhas de Movimento Forçado (Hostis)
3 falhas consecutivas → abandona a escada.

4.4 Timeout de Summons
120 ciclos de onThink → teleporta para o destino.

5. WAYPOINTS DE SUPERFÍCIE (MIGALHAS)
(Funcionamento mantido da v5.7)
Geração, utilização por summons e hostis bloqueados, atualização de timestamps.

6. MAPA COGNITIVO GLOBAL (GlobalMonsterMap) <a name="mapa-cognitivo"></a>
6.1 Classe e Estruturas
GlobalMonsterMap é um singleton que mantém:

Um mapa m_tiles (unordered_map<Position, TileInfo>) – tiles caminháveis conhecidos.

Um mapa m_transitions (unordered_map<Position, vector<Position>>) – arestas de transição de piso (origem → destinos).

cpp
class GlobalMonsterMap {
public:
    static GlobalMonsterMap& getInstance();
    void addTile(const Position& pos, bool walkable, bool dynamic = false);
    bool isWalkable(const Position& pos) const;
    size_t scanArea(const Position& origin, int radius, int maxZUp = 1, int maxZDown = 1, size_t maxTiles = 2000);
    std::vector<Position> findPath(const Position& start, const Position& goal) const;
    void addStairTransition(const Position& from, const Position& to);
    std::vector<Position> getDestinations(const Position& pos);
private:
    std::unordered_map<Position, TileInfo> m_tiles;
    std::unordered_map<Position, std::vector<Position>> m_transitions;
};
6.2 Alimentação do Grafo
Transições do jogador: Em tryUseGlobalMap, todas as StairTransition da fila do líder são injetadas no grafo via addStairTransition, que marca ambos os extremos como caminháveis.

Scan da vizinhança: Quando um monstro precisa e a sua posição não está mapeada, chama scanArea. Este método faz uma BFS limitada, adicionando tiles adjacentes não bloqueados e seguindo as transições já conhecidas para explorar pisos diferentes. O scan pára quando atinge maxTiles (2000 por omissão) ou os limites de raio/Z.

6.3 BFS Multi‑Piso (findPath)
O BFS expande apenas tiles marcados como walkable, mas permite que o objetivo seja uma escada mesmo que esta esteja bloqueada no grafo (verifica TILESTATE_FLOORCHANGE no tile real).
As transições de piso são consideradas como arestas: se o nó atual tem uma transição, os destinos são adicionados à fila desde que estejam no grafo como caminháveis.
Limite de 500 nós expandidos para garantir performance.

6.4.3 Função tryUseGlobalMap
Verifica condições (distância à escada > 2, throttle de 2s).

Obtém as transições do jogador e injeta‑as no GlobalMonsterMap.

Se a posição atual não está mapeada, faz scanArea com raio 15 (e raio 25 se o primeiro falhar).

Chama findPath entre a posição atual e m_stairOrigin.

Se encontra caminho, guarda‑o em m_globalPath e ativa m_usingGlobalPath.

6.4.4 Seguimento da rota global (followGlobalPathStep)
A cada ciclo enquanto m_usingGlobalPath:

Se já chegou à escada (dist ≤ 1) ou o índice ultrapassou o tamanho, limpa a rota.

Caso contrário, move‑se um tile na direção do próximo ponto da rota.

Se o movimento falhar (posição não muda), tenta movimento sem flag FLAG_NOLIMIT; se continuar a falhar, cancela a rota global.

6.4.5 Cancelamento da rota global
A rota global é cancelada automaticamente quando:

O líder fica acessível (verificação no início do onThink_async).

O monstro se aproxima da escada (dist <= 1).

O movimento falha duas vezes consecutivas.

clearGlobalPath() limpa o vetor e repõe os índices.

11. EXEMPLOS DE COMPORTAMENTO
11.1 Hostil usa mapa cognitivo para contornar obstáculo
Jogador percorre um corredor em “L”, largando waypoints.
Rato tenta seguir a escada mas fica preso porque o corredor é bloqueado para um caminho direto.
Após 8 ciclos preso, tryUseGlobalMap() é chamado: o BFS encontra um caminho alternativo através dos tiles já mapeados, que pode incluir uma rota mais longa mas transitável. O rato segue a rota global, contorna o obstáculo e chega à escada.

11.2 Cancelamento da rota global quando líder fica acessível
Durante a perseguição, o rato está a meio de uma rota global.
O líder desce de volta ao mesmo piso e fica a 3 tiles de distância.
O onThink_async deteta que o líder está acessível, cancela a rota global e restaura o follow normal.

12. MANUTENÇÃO E EXPANSÃO FUTURA
12.1 Ajuste de parâmetros do mapa cognitivo
Limite de nós do BFS: alterar maxNodes em findPath.

Tamanho máximo do scan: maxTiles em scanArea.

#include "global_monster_map.hpp"
#include "game.h"
#include "tile.h"

GlobalMonsterMap& GlobalMonsterMap::getInstance() {
    static GlobalMonsterMap instance;
    return instance;
}

void GlobalMonsterMap::addTile(const Position& pos, bool walkable, bool dynamic) {
    m_tiles[pos] = {walkable, dynamic, getCurrentTimeMs()};
}

bool GlobalMonsterMap::isWalkable(const Position& pos) const {
    auto it = m_tiles.find(pos);
    return it != m_tiles.end() && it->second.walkable;
}

void GlobalMonsterMap::addStairTransition(const Position& from, const Position& to) {
    m_transitions[from].push_back(to);
    if (m_tiles.find(from) == m_tiles.end()) m_tiles[from] = {true, false, getCurrentTimeMs()};
    if (m_tiles.find(to) == m_tiles.end())   m_tiles[to]   = {true, false, getCurrentTimeMs()};
}

std::vector<Position> GlobalMonsterMap::getDestinations(const Position& pos) {
    auto it = m_transitions.find(pos);
    if (it != m_transitions.end()) return it->second;
    return {};
}

size_t GlobalMonsterMap::scanArea(const Position& origin, int radius, int maxZUp, int maxZDown, size_t maxTiles) {
    std::queue<Position> queue;
    size_t added = 0;

    auto addPos = [&](const Position& pos) {
        if (m_tiles.find(pos) != m_tiles.end()) return;
        auto tile = g_game().map.getTile(pos);
        if (!tile) return;
        bool blocked = tile->hasFlag(TILESTATE_BLOCKPATH) || tile->hasFlag(TILESTATE_BLOCKSOLID);
        addTile(pos, !blocked, false);
        added++;
        queue.push(pos);
    };

    if (m_tiles.find(origin) == m_tiles.end()) addPos(origin);
    else queue.push(origin);

    while (!queue.empty() && added < maxTiles) {
        Position current = queue.front(); queue.pop();
        if (std::abs(current.x - origin.x) > radius || std::abs(current.y - origin.y) > radius) continue;
        if (current.z < origin.z - maxZDown || current.z > origin.z + maxZUp) continue;

        static const int dx[] = {-1, 1, 0, 0};
        static const int dy[] = {0, 0, -1, 1};
        for (int i = 0; i < 4; ++i) {
            Position next(current.x + dx[i], current.y + dy[i], current.z);
            if (m_tiles.find(next) == m_tiles.end()) addPos(next);
        }

        for (const auto& dest : getDestinations(current)) {
            if (m_tiles.find(dest) == m_tiles.end()) addPos(dest);
        }
    }
    return added;
}

std::vector<Position> GlobalMonsterMap::findPath(const Position& start, const Position& goal) const {
    if (start == goal) return {start};

    bool goalIsStair = false;
    if (auto goalTile = g_game().map.getTile(goal)) {
        goalIsStair = goalTile->hasFlag(TILESTATE_FLOORCHANGE);
    }

    std::unordered_map<Position, Position> cameFrom;
    std::queue<Position> queue;
    queue.push(start);
    cameFrom[start] = start;

    size_t nodes = 0;
    const size_t maxNodes = 500;

    while (!queue.empty() && nodes < maxNodes) {
        Position current = queue.front(); queue.pop();
        if (current == goal) {
            std::vector<Position> path;
            while (current != start) { path.push_back(current); current = cameFrom[current]; }
            std::reverse(path.begin(), path.end());
            return path;
        }

        static const int dx[] = {-1, 1, 0, 0};
        static const int dy[] = {0, 0, -1, 1};
        for (int i = 0; i < 4; ++i) {
            Position next(current.x + dx[i], current.y + dy[i], current.z);
            if (cameFrom.find(next) != cameFrom.end()) continue;
            bool reachable = isWalkable(next);
            if (!reachable && next == goal && goalIsStair) reachable = true;
            if (reachable) { cameFrom[next] = current; queue.push(next); nodes++; }
        }

        auto transIt = m_transitions.find(current);
        if (transIt != m_transitions.end()) {
            for (const auto& dest : transIt->second) {
                if (cameFrom.find(dest) == cameFrom.end() && isWalkable(dest)) {
                    cameFrom[dest] = current;
                    queue.push(dest);
                    nodes++;
                }
            }
        }
    }
    return {};
}

void Monster::tryUseGlobalMap() {
    if (m_usingGlobalPath || !m_pursuingStairs) return;
    Position myPos = getPosition();
    int dist = std::abs(myPos.x - m_stairOrigin.x) + std::abs(myPos.y - m_stairOrigin.y);
    if (dist <= 2) return;

    uint64_t now = getCurrentTimeMs();
    if (now - m_lastGlobalScanAttempt < 2000) return;
    m_lastGlobalScanAttempt = now;

    auto leader = isSummon() ? getMaster() : (!m_attackedCreature.expired() ? m_attackedCreature.lock() : nullptr);
    if (!leader) return;
    auto player = leader->getPlayer();
    if (!player) return;

    // Injetar transições do jogador
    for (const auto& trans : player->getStairTransitions()) {
        g_globalMonsterMap().addStairTransition(trans.origin, trans.destination);
    }

    if (!g_globalMonsterMap().isWalkable(myPos)) {
        g_globalMonsterMap().scanArea(myPos, 15, 1, 1);
    }

    auto path = g_globalMonsterMap().findPath(myPos, m_stairOrigin);
    if (!path.empty()) {
        g_logger().info("[MCR-COGNI] {} rota global encontrada com {} passos.", getName(), path.size());
        m_globalPath = path;
        m_globalPathIndex = 0;
        m_usingGlobalPath = true;
        return;
    }

    // Expansão do scan
    g_globalMonsterMap().scanArea(myPos, 25, 1, 1);
    path = g_globalMonsterMap().findPath(myPos, m_stairOrigin);
    if (!path.empty()) {
        g_logger().info("[MCR-COGNI] {} rota global encontrada (scan expandido) com {} passos.", getName(), path.size());
        m_globalPath = path;
        m_globalPathIndex = 0;
        m_usingGlobalPath = true;
    }
}

void Monster::followGlobalPathStep() {
    if (!m_pursuingStairs || !m_usingGlobalPath || m_globalPathIndex >= m_globalPath.size()) {
        clearGlobalPath();
        return;
    }
    Position myPos = getPosition();
    if (myPos == m_stairOrigin || std::abs(myPos.x - m_stairOrigin.x) + std::abs(myPos.y - m_stairOrigin.y) <= 1) {
        clearGlobalPath();
        return;
    }
    Position nextPos = m_globalPath[m_globalPathIndex];
    if (myPos == nextPos) {
        m_globalPathIndex++;
        if (m_globalPathIndex >= m_globalPath.size()) { clearGlobalPath(); return; }
        nextPos = m_globalPath[m_globalPathIndex];
    }
    Direction dir = getDirectionTo(myPos, nextPos);
    if (dir == DIRECTION_NONE) { clearGlobalPath(); return; }

    Position oldPos = myPos;
    g_game().internalMoveCreature(static_self_cast<Monster>(), dir, FLAG_NOLIMIT);
    if (getPosition() == oldPos) {
        g_game().internalMoveCreature(static_self_cast<Monster>(), dir, 0);
        if (getPosition() == oldPos) clearGlobalPath();
    }
}

void Monster::clearGlobalPath() {
    m_globalPath.clear();
    m_globalPathIndex = 0;
    m_usingGlobalPath = false;
    m_globalScanExpanded = false;
}

    if (m_clearingStairs) return;
    m_clearingStairs = true;
    m_pursuingStairs = false;

    if (m_savedFollowId != 0) {

        if (saved) setFollowCreature(saved);
        m_savedFollowId = 0;
    }

}

int SPAFunctions::luaMonsterScanGlobalArea(lua_State* L) {
    auto monster = Lua::getUserdataShared<Monster>(L, 1, "Monster");
    if (!monster) { lua_pushnumber(L, 0); return 1; }
    int radius = Lua::getNumber<int>(L, 2, 15);
    int zRange = Lua::getNumber<int>(L, 3, 2);
    size_t added = g_globalMonsterMap().scanArea(monster->getPosition(), radius, zRange, zRange);
    lua_pushnumber(L, static_cast<lua_Number>(added));
    return 1;
}

int SPAFunctions::luaMonsterFindGlobalPath(lua_State* L) {
    auto monster = Lua::getUserdataShared<Monster>(L, 1, "Monster");
    if (!monster) { lua_newtable(L); return 1; }
    Position target = Lua::getPosition(L, 2);
    auto path = g_globalMonsterMap().findPath(monster->getPosition(), target);
    lua_newtable(L);
    int idx = 1;
    for (const auto& pos : path) {
        lua_newtable(L);
        Lua::setField(L, "x", pos.x);
        Lua::setField(L, "y", pos.y);
        Lua::setField(L, "z", pos.z);
        lua_rawseti(L, -2, idx++);
    }
    return 1;
}

int SPAFunctions::luaMonsterGetGlobalMapInfo(lua_State* L) {
    auto monster = Lua::getUserdataShared<Monster>(L, 1, "Monster");
    lua_newtable(L);
    if (monster) {
        Lua::setField(L, "tileCount", static_cast<lua_Number>(g_globalMonsterMap().getTiles().size()));
        Lua::setField(L, "currentWalkable", g_globalMonsterMap().isWalkable(monster->getPosition()));
    }
    return 1;
}
