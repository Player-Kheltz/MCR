🎯 OBJETIVO DESTE DOCUMENTO
Este documento descreve a versão estável e completa do sistema de perseguição multi‑piso do Projeto MCR, incluindo todos os ficheiros de código relevantes. Após extensos testes, refinamentos e a incorporação de correcções de estabilidade, o sistema atual é 100% estável, não causa crashes, e fornece perseguição inteligente através de múltiplos mecanismos seguros que executam exclusivamente dentro do ciclo de pensamento de cada criatura.

Relativamente à versão anterior, esta versão introduz as seguintes melhorias:

Revalidação de obstáculos temporários – os tiles bloqueados por Magic Walls, campos, etc. são agora marcados como dinâmicos com um TTL curto e revalidados automaticamente (tanto por limpeza periódica como por validação preguiçosa em cada consulta).

Distinção entre bloqueios permanentes e temporários – a função invalidateTileBlocked usa as flags do tile para separar paredes sólidas de obstáculos removíveis, optimizando a gestão do grafo.

Verificação de direcção no atalho cognitivo – o monstro não tenta subir por um buraco; a direcção da escada é comparada com a direcção desejada antes de iniciar a perseguição.

Segunda tentativa de scan expandido – quando o BFS reactivo falha com um scan de 500 tiles, o monstro pode, passados 10 segundos, tentar um scan maior (1000 tiles) antes de desistir.

Persistência optimizada – o saveToFile ignora tiles dinâmicos, garantindo que o ficheiro de cache não acumula lixo temporário.

Optimização de concorrência – o GlobalMonsterMap utiliza std::shared_mutex, permitindo leituras concorrentes sem bloqueios entre si.

Auto‑reparação do grafo – ao carregar o ficheiro de cache, as transições são validadas e as inválidas removidas automaticamente, garantindo que o sistema se corrige sozinho após qualquer anomalia.

Exploração passiva – os monstros alimentam o grafo com scans periódicos de raio 30, garantindo que as escadas sejam descobertas mesmo sem intervenção do jogador.

GPS directo (tryDirectGps) – removido para estabilidade; a perseguição proactiva e o atalho cognitivo cobrem todas as necessidades.

Teleporte seguro (mcrSafeTeleport) – implementado com busca de tile livre em anel (função findFreeTileNear), evitando teleportes sobre o jogador.

CÓDIGOS

1. VISÃO GERAL
O sistema de perseguição multi‑piso permite que criaturas hostis e summons persigam um jogador através de escadas, buracos, teleportes e rampas. A base do sistema é uma fila de transições de escada (StairTransition) que o jogador deixa ao mudar de piso. As criaturas consultam essa fila e caminham até à escada correspondente.

Novidades na v7.2 (face à v7.1):

Revalidação de obstáculos temporários: os tiles bloqueados por Magic Walls, campos, etc. são agora marcados como dinâmicos com um TTL curto (15 segundos). A função cleanupTemporaryObstacles percorre periodicamente a lista de obstáculos expirados e revalida o estado real do tile. Adicionalmente, o método isWalkableUnlocked realiza validação preguiçosa: sempre que um tile dinâmico é consultado, o motor verifica o estado real do tile no jogo, permitindo que o monstro reaja imediatamente ao desaparecimento de um obstáculo.

Distinção entre bloqueios permanentes e temporários: a função invalidateTileBlocked agora separa tiles sólidos (BLOCKSOLID) – que são marcados como permanentemente bloqueados – de tiles apenas com BLOCKPATH, que são tratados como obstáculos temporários. Isto evita que paredes sejam desnecessariamente revalidadas e reduz o número de tiles dinâmicos no grafo.

Verificação de direcção no atalho cognitivo: o bloco de atalho em checkStairPursuit compara o delta Z da escada com o delta Z do líder, impedindo que um monstro tente usar uma escada que o afaste do alvo (ex.: subir por um buraco).

Segunda tentativa de scan expandido: o tryUseGlobalMap (BFS reactivo) tenta primeiro um scan de 500 tiles; se falhar, agenda uma segunda tentativa com um scan maior (1000 tiles) com um throttle independente de 10 segundos. Isto reduz falsos negativos em áreas densas ou mal mapeadas.

Persistência optimizada: o saveToFile já filtrava tiles dinâmicos; o comportamento foi mantido e documentado.

Optimização de concorrência: o GlobalMonsterMap utiliza std::shared_mutex, permitindo leituras concorrentes sem bloqueios entre si.

Auto‑reparação do grafo: ao carregar o ficheiro de cache (global_monster_map.bin), o sistema chama validateTransitions(), que remove automaticamente transições para tiles inexistentes ou bloqueados, garantindo que o grafo nunca fica corrompido.

Toda a lógica continua a ser executada dentro do onThinkMCR, sem threads externas, garantindo estabilidade total.

2. ARQUITECTURA DA MÁQUINA DE ESTADOS
O comportamento do monstro é regido por uma máquina de estados executada a cada ciclo de onThinkMCR:

Prioridade	Estado	Gatilho	Acção
1	Combate	Jogador visível e atacável	Cancela tudo, faz follow e ataca.
2	Retorno ao spawn/casa	m_returningToSpawn ou m_returningHome activo	Processa retorno (caminhando ou teleportando).
3	Seguir rota global (BFS)	m_usingGlobalPath == true	Segue o caminho planeado; cancela se o líder ficar acessível.
4	Perseguição activa por transições	m_pursuingStairs == true	Tenta alcançar m_stairOrigin. Usa BFS reactivo para desbloquear.
5	Procurar próxima transição	m_pursuingStairs == false, existem transições na fila do jogador	checkStairPursuit(): primeiro tenta atalho pelo mapa cognitivo (com verificação de direcção), depois escolhe a melhor escada da fila do jogador.
6	Perseguição proactiva	Nenhuma transição no piso actual, throttle OK	tryProactiveGlobalPursuit(): usa BFS para encontrar uma escada que leve ao jogador.
7	Exploração passiva	Sempre que ocioso (throttle 30s)	tryExploreArea(): faz scan de raio 30 para alimentar o grafo.

4. MECANISMOS DE TIMEOUT E DESISTÊNCIA
4.1 Timeout Adaptativo (Hostis)
Deadline base: 25 segundos.

Fórmula: deadline = currentTime + (25000 + engagement × 350) ms.

Máximo: 45 segundos (engagement = 100).

Enquanto o líder está visível, o deadline é constantemente renovado.

4.2 Timeout por Perda de Visão
Tempo máximo sem ver o líder: 10 segundos.

Contador m_sightLostTicks incrementa quando o líder não está visível.

Após 10 ticks, o monstro desiste e inicia retorno ao spawn.

4.3 Timeout de Progresso
Se o monstro não reduzir a distância à escada alvo por 4 segundos, abandona a escada e avança para a próxima transição.

4.4 Falhas de Movimento Forçado
3 falhas consecutivas de internalMoveCreature → abandona a escada actual.

4.5 Timeout de Summons
120 ciclos de onThink → teleporta para o destino.

5. WAYPOINTS DE SUPERFÍCIE (MIGALHAS)
Os waypoints são usados como fallback quando o BFS não encontra rota. O jogador larga pontos a cada 5 tiles percorridos, que expiram após 2 minutos sem uso. Summons e hostis podem usar waypoints para encontrar o líder quando o pathfinding falha.

6. MAPA COGNITIVO GLOBAL (GlobalMonsterMap)
O GlobalMonsterMap é um grafo partilhado que armazena tiles caminháveis e transições de piso.

6.1 Alimentação do Grafo
Transições do jogador: injectadas directamente no grafo (addStairTransition).

Scans periódicos: scanArea faz flood‑fill limitado, adicionando tiles caminháveis e transições.

Alimentação contínua: Monster::onWalkComplete adiciona o tile actual.

Invalidação reactiva: quando um movimento falha, invalidateTileBlocked actualiza o grafo, distinguindo bloqueios permanentes de temporários.

Exploração passiva: tryExploreArea faz scans de raio 30 a cada 30 segundos, descobrindo tiles e escadas.

6.2 Persistência e Auto‑Reparação
Gravação: ocorre a cada saveServer, em thread separada, para data/world/global_monster_map.bin. Apenas tiles estáticos (dynamic == false) são salvos. Antes de gravar, validateTransitions() limpa transições inválidas.

Carregamento: no arranque do servidor (Game::loadMap), o ficheiro é lido e, de seguida, validateTransitions() remove automaticamente quaisquer transições para tiles inexistentes ou bloqueados. Isto garante que o grafo nunca permanece corrompido entre sessões.

Limpeza de emergência: se o ficheiro estiver corrompido ao ponto de causar crash, basta apagar o .bin e o servidor iniciará com o grafo vazio, reconstruindo‑o progressivamente.

6.3 Invalidação Reactiva e Revalidação de Obstáculos Temporários
A função invalidateTileBlocked determina se o bloqueio é permanente (BLOCKSOLID) ou temporário (BLOCKPATH sem BLOCKSOLID). Para bloqueios temporários, chama addTemporaryObstacle com um TTL de 15 segundos.

addTemporaryObstacle marca o tile como walkable = false, dynamic = true e regista o timestamp de expiração.

cleanupTemporaryObstacles (chamada a cada 10 segundos) percorre a lista de obstáculos expirados e revalida o tile real, repondo o estado correcto.

Validação preguiçosa: isWalkableUnlocked, sempre que consulta um tile marcado como dynamic, lê o estado real do jogo. Isto garante que o monstro reage imediatamente quando um obstáculo desaparece, sem esperar pelo TTL.

6.4 Descoberta Automática de Escadas
Durante o scanArea, para cada tile com flag FLOORCHANGE, é chamado tile->getDestination(). Se o destino for válido, a transição é registada com as coordenadas reais. Caso contrário, assume‑se z±1 na mesma posição (x,y).

Suporta escadas activas, teleportes e rampas (com destino explícito).

6.5 Validação de Transições
O método validateTransitions() é chamado após o carregamento do ficheiro de cache e antes da gravação. Ele percorre todas as transições e remove destinos que não existem ou são bloqueados (paredes), garantindo que o grafo se mantém íntegro mesmo após eventos inesperados.

7. BFS REACTIVO E PERSEGUIÇÃO PROACTIVA
7.1 BFS Reactivo (tryUseGlobalMap)
Throttle: 3 segundos.

Activado quando a distância à escada alvo é > 1 tile.

Primeira tentativa: scan de 500 tiles, raio 15.

Segunda tentativa (se a primeira falhar): scan de 1000 tiles, raio 18, com throttle independente de 10 segundos.

Se encontrar uma rota, define m_usingGlobalPath = true.

7.2 Perseguição Proactiva (tryProactiveGlobalPursuit)
Throttle: 5 segundos (quando não está a perseguir); 10 segundos (quando está em perseguição activa).

Raio de scan: 25 tiles.

Número máximo de monstros próximos: 2.

Procura a escada com menor custo (distância até à escada + distância do destino ao jogador) entre as transições da fila do jogador. Se encontrar, inicia a perseguição por essa escada.

7.3 Exploração Passiva (tryExploreArea)
Throttle: 30 segundos.

Raio de scan: 30 tiles, até 2 pisos acima/abaixo.

Objectivo: alimentar o GlobalMonsterMap com tiles e escadas, mesmo sem perseguição activa.

Só executa se a cobertura de tiles na área for inferior a 60%.

8. ATALHO VIA MAPA COGNITIVO (checkStairPursuit)
Antes de recorrer à fila de transições do jogador, o monstro verifica se o GlobalMonsterMap já contém uma rota completa até ao jogador. Se o primeiro passo dessa rota for uma escada no piso actual e a direcção for coerente, o monstro inicia a perseguição por essa escada, mesmo que o jogador nunca a tenha utilizado.

Verificação de direcção (novo na v7.2): O código calcula deltaZ = stairDest.z - stairOrigin.z e targetDeltaZ = leader.z - current.z. A escada só é aceite se:

deltaZ e targetDeltaZ tiverem o mesmo sinal (ambos positivos = subir, ambos negativos = descer), ou

O destino da escada for exactamente o piso onde o líder está.

9. GPS DIRECTO E TELEPORTE SEGURO
O tryDirectGps foi removido na v7.2 final para garantir estabilidade. Em seu lugar, foram implementados:

findFreeTileNear: procura um tile vazio num raio até 4 tiles, evitando tiles com jogadores.

mcrSafeTeleport: usa findFreeTileNear para teleportar criaturas para um tile seguro, nunca sobre o jogador. Usado por handleSummonMasterTeleport.

tryAlternativeRouteToLeader: activado no bloco 5.5 e 6, tenta planear uma rota BFS completa até ao líder. Se o líder estiver cercado, usa findNearestApproachTo para encontrar o tile mais próximo possível.

13. COMPORTAMENTO ESPERADO DOS MONSTROS
Perseguição multi‑piso: monstros seguem o jogador através de escadas usando a fila de transições.

Contorno de obstáculos: o BFS reactivo encontra caminhos alternativos quando estão bloqueados.

Atalhos proactivos: mesmo sem ver o jogador, um monstro pode planear uma rota para uma escada que o leve ao alvo, usando o mapa cognitivo.

Atalhos via mapa cognitivo: no momento de escolher a escada, o monstro verifica se já conhece uma rota directa (incluindo escadas não percorridas pelo jogador) e utiliza‑a prioritariamente, desde que a direcção seja coerente.

Reacção imediata a obstáculos: se uma Magic Wall desaparece, o monstro detecta a mudança na próxima consulta ao tile (validação preguiçosa), sem esperar pelo ciclo de limpeza.

Maior persistência: devido aos timeouts alargados (10s sem visão, 25s de deadline base), os monstros desistem menos facilmente.

Exploração passiva: os monstros alimentam o grafo automaticamente a cada 30 segundos, garantindo que o terreno é conhecido mesmo sem perseguição.

Estabilidade total: o servidor não apresenta crashes, mesmo com dezenas de monstros activos.

std::atomic<int> g_activeScans(0);

GlobalMonsterMap& GlobalMonsterMap::getInstance() {
    static GlobalMonsterMap instance;
    return instance;
}

void GlobalMonsterMap::addTile(const Position& pos, bool walkable, bool dynamic) {
    std::unique_lock lock(m_mutex);
    addTileUnlocked(pos, walkable, dynamic);
}

void GlobalMonsterMap::addTileUnlocked(const Position& pos, bool walkable, bool dynamic) {
    auto& info = m_tiles[pos];
    info.walkable = walkable;
    info.dynamic = dynamic;
    info.lastUpdated = getCurrentTimeMs();
}

bool GlobalMonsterMap::isWalkable(const Position& pos) const {
    std::shared_lock lock(m_mutex);
    return isWalkableUnlocked(pos);
}

bool GlobalMonsterMap::isWalkableUnlocked(const Position& pos) const {
    auto it = m_tiles.find(pos);
    if (it == m_tiles.end()) return false;
    if (it->second.dynamic) {
        auto tile = g_game().map.getTile(pos);
        bool blocked = tile && (tile->hasFlag(TILESTATE_BLOCKPATH) || tile->hasFlag(TILESTATE_BLOCKSOLID));
        return !blocked;
    }
    return it->second.walkable;
}

std::vector<Position> GlobalMonsterMap::getDestinations(const Position& pos) const {
    std::shared_lock lock(m_mutex);
    return getDestinationsUnlocked(pos);
}

std::vector<Position> GlobalMonsterMap::getDestinationsUnlocked(const Position& pos) const {
    auto it = m_transitions.find(pos);
    if (it != m_transitions.end()) return it->second;
    return {};
}

void GlobalMonsterMap::addStairTransition(const Position& from, const Position& to) {
    std::unique_lock lock(m_mutex);
    m_transitions[from].push_back(to);
    if (m_tiles.find(from) == m_tiles.end())
        addTileUnlocked(from, true, false);
    if (m_tiles.find(to) == m_tiles.end())
        addTileUnlocked(to, true, false);
}

size_t GlobalMonsterMap::scanArea(const Position& origin, int radius,
                                  int maxZUp, int maxZDown, size_t maxTiles,
                                  const std::deque<StairTransition>& playerStairs) {
    if (g_activeScans >= MAX_ACTIVE_SCANS) return 0;
    g_activeScans++;

    std::unique_lock lock(m_mutex);
    static const int dx[] = {-1, 1, 0, 0, -1, -1, 1, 1};
    static const int dy[] = {0, 0, -1, 1, -1, 1, -1, 1};

    std::queue<Position> queue;
    size_t added = 0;

    auto addPos = [&](const Position& pos) {
        if (m_tiles.find(pos) != m_tiles.end()) return;
        auto tile = g_game().map.getTile(pos);
        if (!tile) return;
        bool blocked = tile->hasFlag(TILESTATE_BLOCKPATH) || tile->hasFlag(TILESTATE_BLOCKSOLID);
        addTileUnlocked(pos, !blocked, false);
        added++;
        queue.push(pos);
    };

    for (const auto& trans : playerStairs) {
        if (m_tiles.find(trans.origin) == m_tiles.end()) addPos(trans.origin);
        if (m_tiles.find(trans.destination) == m_tiles.end()) addPos(trans.destination);
        auto& transVec = m_transitions[trans.origin];
        if (std::find(transVec.begin(), transVec.end(), trans.destination) == transVec.end())
            transVec.push_back(trans.destination);
    }

    bool originMapped = (m_tiles.find(origin) != m_tiles.end());
    if (!originMapped) {
        addPos(origin);
    } else {
        size_t scanned = 0;
        constexpr size_t maxSeedScan = 2000;
        for (const auto& [pos, info] : m_tiles) {
            if (++scanned > maxSeedScan) break;
            if (!info.walkable) continue;
            if (std::abs(pos.x - origin.x) > radius || std::abs(pos.y - origin.y) > radius) continue;
            if (pos.z < origin.z - maxZDown || pos.z > origin.z + maxZUp) continue;

            bool hasUnexplored = false;
            for (int i = 0; i < 8 && !hasUnexplored; ++i) {
                Position next(pos.x + dx[i], pos.y + dy[i], pos.z);
                if (m_tiles.find(next) == m_tiles.end()) hasUnexplored = true;
            }
            if (hasUnexplored) queue.push(pos);
        }
        added = 0;
    }

    while (!queue.empty() && added < maxTiles) {
        Position current = queue.front(); queue.pop();
        if (std::abs(current.x - origin.x) > radius || std::abs(current.y - origin.y) > radius) continue;
        if (current.z < origin.z - maxZDown || current.z > origin.z + maxZUp) continue;

        for (int i = 0; i < 8; ++i) {
            Position next(current.x + dx[i], current.y + dy[i], current.z);
            if (m_tiles.find(next) != m_tiles.end()) continue;
            if (i >= 4) {
                Position adj1(current.x + dx[i], current.y, current.z);
                Position adj2(current.x, current.y + dy[i], current.z);
                auto tile1 = g_game().map.getTile(adj1);
                auto tile2 = g_game().map.getTile(adj2);
                bool blocked1 = tile1 && (tile1->hasFlag(TILESTATE_BLOCKPATH) || tile1->hasFlag(TILESTATE_BLOCKSOLID));
                bool blocked2 = tile2 && (tile2->hasFlag(TILESTATE_BLOCKPATH) || tile2->hasFlag(TILESTATE_BLOCKSOLID));
                if (blocked1 && blocked2) continue;
            }
            addPos(next);
        }

        for (const auto& dest : getDestinationsUnlocked(current)) {
            if (m_tiles.find(dest) == m_tiles.end()) addPos(dest);
        }

        if (current != origin) {
            auto tile = g_game().map.getTile(current);
            if (tile && tile->hasFlag(TILESTATE_FLOORCHANGE)) {
                Position dest = tile->getDestination();
                if (dest.x != 0 || dest.y != 0 || dest.z != 0) {
                    auto& trans = m_transitions[current];
                    if (std::find(trans.begin(), trans.end(), dest) == trans.end()) {
                        trans.push_back(dest);
                        if (m_tiles.find(dest) == m_tiles.end()) addPos(dest);
                    }
                } else {
                    auto tryAdd = [&](int newZ) {
                        if (newZ >= 0 && newZ <= 15) {
                            Position destSimple(current.x, current.y, newZ);
                            auto& trans = m_transitions[current];
                            if (std::find(trans.begin(), trans.end(), destSimple) == trans.end()) {
                                trans.push_back(destSimple);
                                if (m_tiles.find(destSimple) == m_tiles.end()) addPos(destSimple);
                            }
                        }
                    };
                    tryAdd(current.z + 1);
                    tryAdd(current.z - 1);
                }
            }
        }
    }

    g_activeScans--;
    return added;
}

std::vector<Position> GlobalMonsterMap::findPath(const Position& start, const Position& goal) const {
    if (start == goal) return {start};

    bool goalIsStair = false;
    if (auto goalTile = g_game().map.getTile(goal))
        goalIsStair = goalTile->hasFlag(TILESTATE_FLOORCHANGE);

    std::unordered_map<Position, Position> cameFrom;
    std::queue<Position> queue;
    queue.push(start);
    cameFrom[start] = start;

    size_t nodes = 0;
    const size_t maxNodes = 2000;

    while (!queue.empty() && nodes < maxNodes) {
        Position current = queue.front(); queue.pop();
        if (current == goal) {
            std::vector<Position> path;
            for (Position cur = goal; cur != start; ) {
                path.push_back(cur);
                auto it = cameFrom.find(cur);
                if (it == cameFrom.end() || it->second == cur) break;
                cur = it->second;
            }
            std::reverse(path.begin(), path.end());
            return path;
        }

        static const int dx[] = {-1, 1, 0, 0, -1, -1, 1, 1};
        static const int dy[] = {0, 0, -1, 1, -1, 1, -1, 1};
        for (int i = 0; i < 8; ++i) {
            Position next(current.x + dx[i], current.y + dy[i], current.z);
            if (cameFrom.find(next) != cameFrom.end()) continue;

            if (i >= 4) {
                Position adj1(current.x + dx[i], current.y, current.z);
                Position adj2(current.x, current.y + dy[i], current.z);
                bool blocked1 = !isWalkable(adj1) && m_tiles.find(adj1) != m_tiles.end();
                bool blocked2 = !isWalkable(adj2) && m_tiles.find(adj2) != m_tiles.end();
                if (blocked1 && blocked2) continue;
            }

            bool reachable = isWalkable(next);
            if (!reachable && next == goal && goalIsStair) reachable = true;
            if (reachable) {
                cameFrom[next] = current;
                queue.push(next);
                nodes++;
            }
        }

        for (const auto& dest : getDestinations(current)) {
            if (cameFrom.find(dest) == cameFrom.end() && isWalkable(dest)) {
                cameFrom[dest] = current;
                queue.push(dest);
                nodes++;
            }
        }
    }
    return {};
}

std::vector<Position> GlobalMonsterMap::findNearestApproachTo(const Position& start, const Position& goal) const {
    std::unordered_map<Position, Position> cameFrom;
    std::queue<Position> queue;
    queue.push(start);
    cameFrom[start] = start;

    Position bestTile = start;
    int bestDist = std::abs(start.x - goal.x) + std::abs(start.y - goal.y);

    size_t nodes = 0;
    const size_t maxNodes = 2000;

    while (!queue.empty() && nodes < maxNodes) {
        Position current = queue.front(); queue.pop();

        int dist = std::abs(current.x - goal.x) + std::abs(current.y - goal.y);
        if (dist < bestDist) {
            auto tile = g_game().map.getTile(current);
            if (tile && !tile->getTopCreature()) {
                bestDist = dist;
                bestTile = current;
            }
        }

        static const int dx[] = {-1, 1, 0, 0};
        static const int dy[] = {0, 0, -1, 1};
        for (int i = 0; i < 4; ++i) {
            Position next(current.x + dx[i], current.y + dy[i], current.z);
            if (cameFrom.find(next) != cameFrom.end()) continue;
            auto tile = g_game().map.getTile(next);
            if (!tile || !isWalkable(next) || tile->hasFlag(TILESTATE_BLOCKPATH)) continue;
            if (next != goal && tile->getTopCreature()) continue;
            cameFrom[next] = current;
            queue.push(next);
            nodes++;
        }
    }

    if (bestTile == start) return {};

    std::vector<Position> path;
    Position cur = bestTile;
    while (cur != start) {
        path.push_back(cur);
        auto it = cameFrom.find(cur);
        if (it == cameFrom.end() || it->second == cur) break;
        cur = it->second;
    }
    std::reverse(path.begin(), path.end());
    return path;
}

void GlobalMonsterMap::notifyTileChange(const Position& pos) {
    std::unique_lock lock(m_mutex);
    auto tile = g_game().map.getTile(pos);
    bool blocked = tile ? (tile->hasFlag(TILESTATE_BLOCKPATH) || tile->hasFlag(TILESTATE_BLOCKSOLID)) : true;
    auto it = m_tiles.find(pos);
    if (it != m_tiles.end() && it->second.walkable == !blocked) return;
    addTileUnlocked(pos, !blocked, it != m_tiles.end() && it->second.dynamic);
}

void GlobalMonsterMap::addTemporaryObstacle(const Position& pos, uint64_t durationMs) {
    std::unique_lock lock(m_mutex);
    m_temporaryObstacles[pos] = {getCurrentTimeMs() + durationMs};
    auto& info = m_tiles[pos];
    info.walkable = false;
    info.dynamic = true;
    info.lastUpdated = getCurrentTimeMs();
}

void GlobalMonsterMap::cleanupTemporaryObstacles() {
    std::unique_lock lock(m_mutex);
    uint64_t now = getCurrentTimeMs();
    for (auto it = m_temporaryObstacles.begin(); it != m_temporaryObstacles.end(); ) {
        if (now >= it->second.expiryTime) {
            auto tile = g_game().map.getTile(it->first);
            bool walkable = true;
            if (tile) {
                walkable = !(tile->hasFlag(TILESTATE_BLOCKPATH) || tile->hasFlag(TILESTATE_BLOCKSOLID));
            }
            auto tileIt = m_tiles.find(it->first);
            if (tileIt != m_tiles.end()) {
                tileIt->second.walkable = walkable;
                tileIt->second.dynamic = false;
                tileIt->second.lastUpdated = now;
            }
            it = m_temporaryObstacles.erase(it);
        } else {
            ++it;
        }
    }
}

void GlobalMonsterMap::saveToFile(const std::string& filename) {
    // Validar antes de guardar
    validateTransitions();

    std::vector<std::tuple<uint16_t, uint16_t, uint8_t, uint16_t>> staticTiles;
    {
        std::shared_lock lock(m_mutex);
        for (const auto& [pos, info] : m_tiles) {
            if (!info.dynamic) {
                uint16_t flags = info.walkable ? 0x01 : 0x00;
                staticTiles.emplace_back(pos.x, pos.y, pos.z, flags);
            }
        }
    }
    std::thread([staticTiles, filename]() {
        std::ofstream file(filename, std::ios::binary);
        if (!file) return;
        uint32_t count = static_cast<uint32_t>(staticTiles.size());
        file.write(reinterpret_cast<const char*>(&count), sizeof(count));
        for (const auto& [x, y, z, flags] : staticTiles) {
            file.write(reinterpret_cast<const char*>(&x), sizeof(x));
            file.write(reinterpret_cast<const char*>(&y), sizeof(y));
            file.write(reinterpret_cast<const char*>(&z), sizeof(z));
            file.write(reinterpret_cast<const char*>(&flags), sizeof(flags));
        }
    }).detach();
}

void GlobalMonsterMap::loadFromFile(const std::string& filename) {
    std::ifstream file(filename, std::ios::binary);
    if (!file) return;
    uint32_t count;
    file.read(reinterpret_cast<char*>(&count), sizeof(count));
    for (uint32_t i = 0; i < count; ++i) {
        uint16_t x, y, flags;
        uint8_t z;
        file.read(reinterpret_cast<char*>(&x), sizeof(x));
        file.read(reinterpret_cast<char*>(&y), sizeof(y));
        file.read(reinterpret_cast<char*>(&z), sizeof(z));
        file.read(reinterpret_cast<char*>(&flags), sizeof(flags));
        bool walkable = (flags & 0x01) != 0;
        addTile(Position(x, y, z), walkable, false);
    }
}

void GlobalMonsterMap::clear() {
    std::unique_lock lock(m_mutex);
    m_tiles.clear();
    m_transitions.clear();
    m_temporaryObstacles.clear();
}

void GlobalMonsterMap::validateTransitions() {
    std::unique_lock lock(m_mutex);
    for (auto it = m_transitions.begin(); it != m_transitions.end(); ) {
        auto& dests = it->second;
        dests.erase(
            std::remove_if(dests.begin(), dests.end(), [this](const Position& dest) {
                auto tile = g_game().map.getTile(dest);
                if (!tile) return true;
                if (tile->hasFlag(TILESTATE_FLOORCHANGE)) return false;
                return tile->hasFlag(TILESTATE_BLOCKPATH) || tile->hasFlag(TILESTATE_BLOCKSOLID);
            }),
            dests.end()
        );

        if (dests.empty()) {
            it = m_transitions.erase(it);
        } else {
            ++it;
        }
    }
}

void Monster::onThinkMCR() {
    if (m_inOnThinkMCR) return;
    m_inOnThinkMCR = true;

    static uint64_t lastCleanup = 0;
    if (getCurrentTimeMs() - lastCleanup > 10000) {
        g_globalMonsterMap().cleanupTemporaryObstacles();
        lastCleanup = getCurrentTimeMs();
    }

    // ═══ 0. Seguir rota global (BFS), apenas se estiver ativa ═══
    if (m_usingGlobalPath) {
        std::shared_ptr<Creature> leader;
        if (isSummon()) {
            auto master = getMaster();
            if (master && master.get()) leader = master;
        } else {
            auto attacked = !m_attackedCreature.expired() ? m_attackedCreature.lock() : nullptr;
            if (attacked && attacked.get()) leader = attacked;
        }
        if (leader && leader.get() && !leader->isRemoved()) {
            if (leader->getPosition().z == getPosition().z) {
                int distToLeader = std::abs(getPosition().x - leader->getPosition().x)
                                 + std::abs(getPosition().y - leader->getPosition().y);
                if (distToLeader <= 3) {
                    clearGlobalPath();
                    setFollowCreature(leader);
                    setAttackedCreature(leader);
                }
            }
            if (m_usingGlobalPath) {
                FindPathParams fpp;
                getPathSearchParams(getMonster(), fpp);
                std::vector<Direction> dummy;
                if (getPathTo(leader->getPosition(), dummy, fpp)) {
                    clearGlobalPath();
                    setFollowCreature(leader);
                    setAttackedCreature(leader);
                }
            }
        }
        if (m_usingGlobalPath) {
            followGlobalPathStep();
            m_inOnThinkMCR = false;
            return;
        }
    }

    // ═══ 1. Combate – prioridade máxima ═══
    if (!isSummon() && !m_returningToSpawn && !m_returningHome) {
        auto targetCreature = getAttackedCreature();
        if (!targetCreature) targetCreature = getFollowCreature();
        if (targetCreature && targetCreature.get() && !targetCreature->isRemoved() && targetCreature->getPlayer()
            && canUseAttack(getPosition(), targetCreature)) {
            m_hasReturnedToSpawn = false;
            clearStairPursuit();
            clearGlobalPath();
            setFollowCreature(targetCreature);
            setAttackedCreature(targetCreature);
        }
    }

    // ═══ 2. Retorno ao spawn / casa ═══
    if (m_returningHome) {
        if (m_defensiveRetaliation) {
            if (getCurrentTimeMs() - m_lastRetaliationTime > 5000 || !getAttackedCreature()) {
                m_defensiveRetaliation = false;
                setFollowCreature(nullptr);
                setAttackedCreature(nullptr);
            } else { m_inOnThinkMCR = false; return; }
        }
        processarRetornoSimples();
        m_inOnThinkMCR = false;
        return;
    }
    if (m_returningToSpawn) {
        if (m_defensiveRetaliation) {
            if (getCurrentTimeMs() - m_lastRetaliationTime > 5000 || !getAttackedCreature()) {
                m_defensiveRetaliation = false;
                setFollowCreature(nullptr);
                setAttackedCreature(nullptr);
            } else { m_inOnThinkMCR = false; return; }
        }
        processarRetornoSpawn();
        m_inOnThinkMCR = false;
        return;
    }

    // ═══ 3. Summon teleporte abrupto ═══
    // (desativado)

    // ═══ 4. Perseguição por transições ═══
    if (isSummon() || (!isSummon() && (!m_attackedCreature.expired() ||
        (m_persistentTargetPlayer && !m_persistentTargetPlayer->getStairTransitions().empty())))) {
        checkStairPursuit();
    }

    // ═══ 5. Perseguição ativa (m_pursuingStairs == true) ═══
    if (m_pursuingStairs) {
        Position myPos = getPosition();
        int dist = std::abs(myPos.x - m_stairOrigin.x) + std::abs(myPos.y - m_stairOrigin.y);
        m_stairTotalCycles++;

        auto targetPlayer = getTargetPlayer();
        bool visible = targetPlayer && canSee(targetPlayer->getPosition());
        updateEngagement(visible, visible, visible);

        uint64_t newDeadline = computePursuitDeadline();
        if (newDeadline > m_pursuitDeadline + 5000) {
            m_pursuitDeadline = newDeadline;
        }

        // Obter líder de forma segura
        std::shared_ptr<Creature> leader;
        if (isSummon()) {
            auto master = getMaster();
            if (master && master.get()) leader = master;
        } else {
            auto attacked = !m_attackedCreature.expired() ? m_attackedCreature.lock() : nullptr;
            if (attacked && attacked.get()) leader = attacked;
        }
        bool leaderVisible = leader && leader.get() && canSee(leader->getPosition());

        // Timeout por perda de visão + deadline
        if (leaderVisible) {
            m_sightLostTicks = 0;
            m_pursuitDeadline = computePursuitDeadline();
        } else {
            m_sightLostTicks++;
            if (m_sightLostTicks >= SIGHT_LOST_TIMEOUT_TICKS) {
                clearStairPursuit();
                if (!isSummon()) iniciarRetornoSpawn();
                m_inOnThinkMCR = false; return;
            }
            if (getCurrentTimeMs() > m_pursuitDeadline) {
                clearStairPursuit();
                if (!isSummon()) iniciarRetornoSpawn();
                else m_nextStairIndex = m_currentTransitionIndex + 1;

                if (getPathTo(leader->getPosition(), dummy, fpp)) shouldCancel = true;

                    FindPathParams fpp;
                    fpp.minTargetDist = 0; fpp.maxTargetDist = 0; fpp.fullPathSearch = false; fpp.clearSight = false; fpp.maxSearchDist = 0;
                    std::vector<Direction> dummy;
                    if (getPathTo(leader->getPosition(), dummy, fpp)) shouldCancel = true;
                }
            }
            if (shouldCancel) {
                if (!isSummon()) m_returnPath.clear();
                m_nextStairIndex = m_currentTransitionIndex + 1;
                clearStairPursuit();
                setFollowCreature(leader);
                m_inOnThinkMCR = false; return;
            }
        }

        // Waypoint de superfície alcançado?
        if (m_followingSurfaceWaypoint && (myPos == m_stairOrigin || dist <= 1)) {
            if (auto leader2 = isSummon() ? getMaster() : nullptr) {
                if (leader2 && leader2.get() && leader2->getPlayer()) leader2->getPlayer()->touchSurfaceWaypoint(m_currentSurfaceWaypointIndex);
            }
            m_currentSurfaceWaypointIndex++;

            m_inOnThinkMCR = false; return;

            }
            if (!isSummon() && leader && leader.get() && leader->getPlayer() && leader->getPosition().z == m_stairDestination.z) {

                if (d <= 3) {
                    setFollowCreature(leader);
                    setAttackedCreature(leader);
                }
            }
            m_nextStairIndex = m_currentTransitionIndex + 1;
            clearStairPursuit();
            m_inOnThinkMCR = false; return;
        }

        // Bloqueio e desencravamento
        if (isWalkingToPosition()) {
            m_stairStuckCount++;

            } else if (m_stairStuckCount >= 8) {

                    if (leader3 && leader3.get()) {
                        auto player = leader3->getPlayer();
                        if (player) {
                            auto& waypoints = player->getSurfaceWaypoints();
                            if (!waypoints.empty()) {
                                const SurfaceWaypoint* bestWp = nullptr;
                                int bestDist = std::numeric_limits<int>::max();
                                for (const auto& wp : waypoints) {
                                    if (wp.pos.z == m_stairOrigin.z) {
                                        int d = std::abs(wp.pos.x - m_stairOrigin.x) + std::abs(wp.pos.y - m_stairOrigin.y);
                                        if (d < bestDist) { bestDist = d; bestWp = &wp; }
                                    }
                                }
                                if (bestWp && bestWp->pos != myPos) {
                                    walkToWaypoint(bestWp->pos);
                                    m_stairStuckCount = 0;
                                    m_inOnThinkMCR = false; return;
                                }
                            }
                        }
                    }

                    std::vector<Position> adjacentTiles;
                    for (int dx = -1; dx <= 1; ++dx)

                        }
                    if (adjacentTiles.empty()) {
                        for (int dx = -1; dx <= 1; ++dx)
                            for (int dy = -1; dy <= 1; ++dy) {
                                if (dx == 0 && dy == 0) continue;
                                Position adjPos(myPos.x + dx, myPos.y + dy, myPos.z);
                                auto tile = g_game().map.getTile(adjPos);
                                if (tile && !tile->hasFlag(TILESTATE_BLOCKPATH)) adjacentTiles.push_back(adjPos);
                            }
                    }
                    if (!adjacentTiles.empty()) {
                        Position bestTile = adjacentTiles.front();
                        int bestDist = std::abs(myPos.x - bestTile.x) + std::abs(myPos.y - bestTile.y);
                        for (const auto& tile : adjacentTiles) {
                            int d = std::abs(myPos.x - tile.x) + std::abs(myPos.y - tile.y);
                            if (d < bestDist) { bestDist = d; bestTile = tile; }
                        }
                        walkToWaypoint(bestTile);
                        m_stairStuckCount = 0;
                        m_inOnThinkMCR = false; return;
                    }
                    m_stairStuckCount = 0;
                    m_inOnThinkMCR = false; return;
                } else {
                    shouldForce = true;
                }
            }

            if (shouldForce) {
                Position oldPos = getPosition();
                Direction dir = getDirectionTo(myPos, m_stairOrigin);
                if (dir != DIRECTION_NONE) {
                    Position nextPos = getNextPosition(dir, myPos);
                    ReturnValue ret = g_game().internalMoveCreature(static_self_cast<Monster>(), dir, 0);
                    if (ret != RETURNVALUE_NOERROR && dist <= 2) {
                        ret = g_game().internalMoveCreature(static_self_cast<Monster>(), dir, FLAG_IGNOREBLOCKITEM);
                    }
                    if (ret != RETURNVALUE_NOERROR) {
                        invalidateTileBlocked(nextPos);
                    }
                }
                if (getPosition() == oldPos) {
                    m_forcedMoveFailCount++;
                } else {
                    m_forcedMoveFailCount = 0;
                    if (!isSummon()) {
                        m_lastDistImprovementTime = getCurrentTimeMs();
                        m_minDistToStair = std::abs(getPosition().x - m_stairOrigin.x) + std::abs(getPosition().y - m_stairOrigin.y);
                    }
                }
                m_stairStuckCount = 0;
                if (m_forcedMoveFailCount >= 3) {
                    m_nextStairIndex = m_currentTransitionIndex + 1;
                    clearStairPursuit();
                    m_inOnThinkMCR = false; return;
                }
                m_inOnThinkMCR = false; return;
            }
            m_inOnThinkMCR = false; return;
        }

        // Aguardar aliados (summons)
        if (isSummon() && dist > 0 && m_stairStuckCount < 3) {
            Direction dirToWaypoint = getDirectionTo(myPos, m_stairOrigin);
            if (dirToWaypoint != DIRECTION_NONE) {
                Position frontPos = getNextPosition(dirToWaypoint, myPos);
                auto frontTile = g_game().map.getTile(frontPos);
                if (frontTile) {
                    auto frontCreature = frontTile->getTopCreature();
                    if (frontCreature) {
                        auto frontMonster = frontCreature->getMonster();
                        if (frontMonster && frontMonster->isSummon() && frontMonster->getMaster() == getMaster()) {
                            m_stairTotalCycles--;
                            m_inOnThinkMCR = false; return;
                        }
                    }
                }
            }
        }

        if (!isSummon() && !isWalkingToPosition()) {
            int currentDist = std::abs(getPosition().x - m_stairOrigin.x) + std::abs(getPosition().y - m_stairOrigin.y);
            if (currentDist < m_minDistToStair) {
                m_minDistToStair = currentDist;
                m_lastDistImprovementTime = getCurrentTimeMs();
            }
        }
        if (!isSummon() && !isWalkingToPosition() && getCurrentTimeMs() - m_lastDistImprovementTime > 4000) {
            m_nextStairIndex = m_currentTransitionIndex + 1;
            clearStairPursuit();
            m_inOnThinkMCR = false; return;
        }

        // ═══ 5.5 Rota alternativa quando bloqueado no mesmo piso ═══
        if (!m_usingGlobalPath && !isWalkingToPosition() && leader && leader.get() && leader->getPosition().z == myPos.z) {
            tryAlternativeRouteToLeader();
            if (m_usingGlobalPath) {
                m_inOnThinkMCR = false;
                return;
            }
        }

        walkToWaypoint(m_stairOrigin);
        if (!isWalkingToPosition()) {
            if (isSummon()) {
                m_stairPathfindFailCount++;
                if (m_stairPathfindFailCount >= 5) {
                    g_game().internalTeleport(static_self_cast<Monster>(), m_stairDestination);
                    m_nextStairIndex = m_currentTransitionIndex + 1;
                    clearStairPursuit();
                }
            }
        } else {
            m_stairPathfindFailCount = 0;
        }
        m_inOnThinkMCR = false;
        return;
    }

    // ═══ 6. Exploração passiva + rota alternativa + perseguição proativa ═══
    if (!m_returningToSpawn && !m_returningHome) {
        // 6a. Alimentar o mapa cognitivo (exploração passiva)
        if (!m_pursuingStairs && !m_usingGlobalPath) {
            tryExploreArea();
        }

        // 6b. Rota alternativa ou perseguição proativa
        if (!m_pursuingStairs) {
            tryAlternativeRouteToLeader();
            if (!m_usingGlobalPath) {
                tryProactiveGlobalPursuit();
            }
        }
    }

    m_inOnThinkMCR = false;
}
Monster::checkStairPursuit() (com atalho e verificação de direcção)
cpp
void Monster::checkStairPursuit() {
    if (m_pursuingStairs || m_returningToSpawn) return;
    if (m_hasReturnedToSpawn && !isSummon()) return;

    std::shared_ptr<Creature> leader;
    if (isSummon()) {
        leader = getMaster();
    } else {
        if (!m_attackedCreature.expired()) leader = m_attackedCreature.lock();
        else if (m_persistentTargetPlayer) leader = m_persistentTargetPlayer;
    }
    if (!leader) return;
    auto leaderPlayer = leader->getPlayer();
    if (!leaderPlayer) return;

    if (!isSummon()) {
        if (m_persistentTargetPlayer != leaderPlayer) m_nextStairIndex = 0;
        else if (!m_persistentTargetPlayer) m_nextStairIndex = 0;
        m_persistentTargetPlayer = leaderPlayer;
    }

    // ═══ Atalho via mapa cognitivo (com verificação de direcção) ═══
    if (!m_pursuingStairs && g_globalMonsterMap().isWalkable(getPosition())) {
        auto path = g_globalMonsterMap().findPath(getPosition(), leader->getPosition());
        if (!path.empty() && path.size() >= 2 && path[0].z != path[1].z) {
            Position stairOrigin = path[0];
            Position stairDest   = path[1];
            if (stairOrigin.z == getPosition().z) {
                int deltaZ = stairDest.z - stairOrigin.z;
                int targetDeltaZ = leader->getPosition().z - getPosition().z;
                bool directionOk = (deltaZ > 0 && targetDeltaZ > 0) ||
                                   (deltaZ < 0 && targetDeltaZ < 0) ||
                                   (stairDest.z == leader->getPosition().z);
                if (directionOk) {
                    auto tile = g_game().map.getTile(stairOrigin);
                    bool isActive = tile && tile->hasFlag(TILESTATE_FLOORCHANGE);
                    m_pursuingStairs = true;
                    clearGlobalPath();
                    m_stairOrigin          = stairOrigin;
                    m_stairDestination     = stairDest;
                    m_stairIsActive        = isActive;
                    m_stairStuckCount      = 0;
                    m_stairTotalCycles     = 0;
                    m_stairPathfindFailCount = 0;
                    m_forcedMoveFailCount  = 0;
                    m_currentTransitionIndex = std::numeric_limits<size_t>::max();
                    m_pursuitStartTime      = getCurrentTimeMs();
                    m_stairAttemptStartTime = getCurrentTimeMs();
                    m_minDistToStair        = std::numeric_limits<int>::max();
                    m_lastDistImprovementTime = getCurrentTimeMs();
                    m_homePosition          = getPosition();
                    m_pursuitDeadline       = computePursuitDeadline();
                    m_sightLostTicks        = 0;
                    m_savedFollowId         = leader->getID();
                    setFollowCreature(nullptr);
                    g_logger().info("[MCR-STAIR] {} encontrou atalho pelo mapa cognitivo: escada em {} para {} (deltaZ={})",
                        getName(), stairOrigin.toString(), stairDest.toString(), deltaZ);
                    walkToWaypoint(m_stairOrigin);
                    return;
                }
            }
        }
    }

    // Código existente: fila de transições do jogador
    auto& fila = leaderPlayer->getStairTransitions();
    if (fila.empty()) {
        if (isSummon()) {
            if (leader->getPosition().z != getPosition().z) {
                g_game().internalTeleport(static_self_cast<Monster>(), leader->getPosition());
            } else {
                FindPathParams fpp;
                getPathSearchParams(getMonster(), fpp);
                std::vector<Direction> dummy;
                if (!getPathTo(leader->getPosition(), dummy, fpp)) {
                    auto& waypoints = leaderPlayer->getSurfaceWaypoints();
                    if (m_currentSurfaceWaypointIndex >= waypoints.size())
                        m_currentSurfaceWaypointIndex = 0;
                    while (m_currentSurfaceWaypointIndex < waypoints.size()) {
                        const SurfaceWaypoint& wp = waypoints[m_currentSurfaceWaypointIndex];
                        if (wp.pos.z == getPosition().z) {
                            m_pursuingStairs = true;
                            clearGlobalPath();
                            m_stairOrigin = wp.pos;
                            m_stairDestination = wp.pos;
                            m_stairIsActive = false;
                            m_stairStuckCount = 0;
                            m_stairTotalCycles = 0;
                            m_stairPathfindFailCount = 0;
                            m_followingSurfaceWaypoint = true;
                            m_forcedMoveFailCount = 0;
                            m_savedFollowId = leader->getID();
                            setFollowCreature(nullptr);
                            walkToWaypoint(m_stairOrigin);
                            return;
                        }
                        ++m_currentSurfaceWaypointIndex;
                    }
                }
            }
        }
        return;
    }

    if (m_nextStairIndex == std::numeric_limits<size_t>::max())
        m_nextStairIndex = isSummon() ? fila.size() : 0;

    // ═══ SELECCIONAR A MELHOR ESCADA (custo combinado) ═══
    size_t idx = m_nextStairIndex;
    int bestCost = std::numeric_limits<int>::max();
    size_t bestIdx = std::numeric_limits<size_t>::max();
    Position playerPos = leader->getPosition();

    while (idx < fila.size()) {
        const StairTransition& trans = fila[idx];
        if (trans.origin.z == getPosition().z) {
            if (leader->getPosition().z == getPosition().z) {
                FindPathParams fpp;
                getPathSearchParams(getMonster(), fpp);
                std::vector<Direction> dummy;
                if (getPathTo(leader->getPosition(), dummy, fpp)) return;
            }
            auto path = g_globalMonsterMap().findPath(getPosition(), trans.origin);
            int distToOrigin = path.empty()
                ? (std::abs(getPosition().x - trans.origin.x) + std::abs(getPosition().y - trans.origin.y))
                : static_cast<int>(path.size());
            int distDestToPlayer = std::abs(trans.destination.x - playerPos.x)
                                 + std::abs(trans.destination.y - playerPos.y);
            int cost = distToOrigin + distDestToPlayer;
            if (cost < bestCost) { bestCost = cost; bestIdx = idx; }
        }
        ++idx;
    }

    if (bestIdx != std::numeric_limits<size_t>::max()) {
        const StairTransition& trans = fila[bestIdx];
        m_pursuingStairs = true;
        clearGlobalPath();
        m_stairOrigin = trans.origin;
        m_stairDestination = trans.destination;
        m_stairIsActive = trans.isActive;
        m_stairStuckCount = 0;
        m_stairTotalCycles = 0;
        m_stairPathfindFailCount = 0;
        m_currentTransitionIndex = bestIdx;
        m_pursuitStartTime = getCurrentTimeMs();
        m_stairAttemptStartTime = getCurrentTimeMs();
        m_minDistToStair = std::numeric_limits<int>::max();
        m_lastDistImprovementTime = getCurrentTimeMs();
        m_homePosition = getPosition();
        m_pursuitDeadline = computePursuitDeadline();
        m_sightLostTicks = 0;
        m_savedFollowId = leader->getID();
        setFollowCreature(nullptr);
        g_logger().info("[MCR-STAIR] {} perseguição INICIADA. stairOrigin={} stairDest={} isActive={} (melhor índice {})",
            getName(), m_stairOrigin.toString(), m_stairDestination.toString(), m_stairIsActive, bestIdx);
        walkToWaypoint(m_stairOrigin);
    }
}
Monster::walkToWaypoint() (com tile adjacente)
cpp
void Monster::walkToWaypoint(const Position& target) {
    Position actualTarget = target;

    if (m_pursuingStairs && target == m_stairOrigin) {
        static const int dx[] = {-1, 1, 0, 0};
        static const int dy[] = {0, 0, -1, 1};
        for (int i = 0; i < 4; ++i) {
            Position adj(target.x + dx[i], target.y + dy[i], target.z);
            auto tile = g_game().map.getTile(adj);
            if (tile && !tile->hasFlag(TILESTATE_BLOCKPATH) && !tile->hasFlag(TILESTATE_BLOCKSOLID)) {
                actualTarget = adj;
                break;
            }
        }
    }

    std::vector<Direction> listDir;
    FindPathParams fpp;
    getPathSearchParams(getMonster(), fpp);
    fpp.minTargetDist = 0;
    fpp.maxTargetDist = 1;
    fpp.fullPathSearch = true;
    fpp.clearSight = false;
    fpp.maxSearchDist = 0;

    if (getPathTo(actualTarget, listDir, fpp) && !listDir.empty()) {
        g_logger().info("[MCR-WALK] {} caminho encontrado: {} passos. Primeiro passo: {}",
            getName(), listDir.size(), static_cast<int>(listDir.front()));
        startAutoWalk(listDir);
        setWalkingToPosition(true);
        if (!isSummon()) {
            m_lastDistImprovementTime = getCurrentTimeMs();
            m_minDistToStair = std::abs(getPosition().x - actualTarget.x) + std::abs(getPosition().y - actualTarget.y);
        }
    } else {
        g_logger().info("[MCR-WALK] {} FALHA no pathfinding, movimento simples na direcção do alvo.", getName());
        Direction dir = getDirectionTo(getPosition(), actualTarget);
        if (dir != DIRECTION_NONE) {
            Position oldPos = getPosition();
            g_game().internalMoveCreature(static_self_cast<Monster>(), dir, 0);
            if (getPosition() == oldPos) {
                g_logger().info("[MCR-WALK] {} movimento forçado não mudou a posição, a limpar walkingToPosition.", getName());
                setWalkingToPosition(false);
            }
        } else {
            g_logger().info("[MCR-WALK] {} sem direcção válida!", getName());
            setWalkingToPosition(false);
        }
    }
}
Monster::tryUseGlobalMap() (BFS reactivo com scan expandido)

void Monster::tryUseGlobalMap() {
    if (m_usingGlobalPath || !m_pursuingStairs) return;
    Position myPos = getPosition();
    int dist = std::abs(myPos.x - m_stairOrigin.x) + std::abs(myPos.y - m_stairOrigin.y);
    if (dist <= 1) return;

    uint64_t now = getCurrentTimeMs();
    if (now - m_lastGlobalScanAttempt < 3000) return;
    m_lastGlobalScanAttempt = now;

    std::shared_ptr<Creature> leader;
    if (isSummon()) leader = getMaster();
    else {
        leader = !m_attackedCreature.expired() ? m_attackedCreature.lock() : nullptr;
        if (!leader) leader = m_persistentTargetPlayer;
    }
    if (!leader) return;
    auto player = leader->getPlayer();
    if (!player) return;

    for (const auto& trans : player->getStairTransitions())
        g_globalMonsterMap().addStairTransition(trans.origin, trans.destination);

    size_t added = 0;
    // Primeira tentativa: scan de 500 tiles
    if (mcrTryScanArea(myPos, 15, 1, 1, 500, player->getStairTransitions(), &added)) {
        if (added > 0) {
            auto path = g_globalMonsterMap().findPath(myPos, m_stairOrigin);
            if (!path.empty()) {
                m_globalPath = path;
                m_globalPathIndex = 0;
                m_usingGlobalPath = true;
                return;
            }
        }
    }

    // Segunda tentativa: scan expandido de 1000 tiles, throttle 10s
    if (now - m_lastExtendedScanAttempt >= 10000) {
        m_lastExtendedScanAttempt = now;
        size_t added2 = 0;
        if (mcrTryScanArea(myPos, 18, 1, 1, 1000, player->getStairTransitions(), &added2)) {
            if (added2 > 0) {
                auto path = g_globalMonsterMap().findPath(myPos, m_stairOrigin);
                if (!path.empty()) {
                    m_globalPath = path;
                    m_globalPathIndex = 0;
                    m_usingGlobalPath = true;
                }
            }
        }
    }
}
Monster::tryProactiveGlobalPursuit() (perseguição proactiva)

void Monster::tryProactiveGlobalPursuit() {
    if (m_usingGlobalPath || m_returningToSpawn || m_returningHome) return;

    uint64_t throttle = m_pursuingStairs ? 10000 : 5000;
    uint64_t now = getCurrentTimeMs();
    if (now - m_lastRouteRequestTime < throttle) return;
    m_lastRouteRequestTime = now;

    if (m_proactiveFailCount < 1) return;

    int nearbyMonsters = 0;
    for (int dx = -5; dx <= 5; ++dx) {
        for (int dy = -5; dy <= 5; ++dy) {
            if (dx == 0 && dy == 0) continue;
            Position pos(getPosition().x + dx, getPosition().y + dy, getPosition().z);
            auto tile = g_game().map.getTile(pos);
            if (tile) {
                auto topCreature = tile->getTopCreature();
                if (topCreature) {
                    auto monster = topCreature->getMonster();
                    if (monster && !monster->isSummon() && monster->getID() != getID()) {
                        nearbyMonsters++;
                        if (nearbyMonsters > 2) break;
                    }
                }
            }
        }
        if (nearbyMonsters > 2) break;
    }
    if (nearbyMonsters > 2) return;

    std::shared_ptr<Creature> leader;
    if (isSummon()) leader = getMaster();
    else {
        leader = !m_attackedCreature.expired() ? m_attackedCreature.lock() : nullptr;
        if (!leader) leader = m_persistentTargetPlayer;
    }
    if (!leader || !leader->getPlayer()) return;

    const auto& stairs = leader->getPlayer()->getStairTransitions();
    if (stairs.empty()) return;

    for (const auto& trans : stairs)
        g_globalMonsterMap().addStairTransition(trans.origin, trans.destination);

    size_t added = 0;
    if (mcrTryScanArea(getPosition(), 25, 1, 1, 500, stairs, &added)) {
        if (added == 0) { m_proactiveFailCount++; return; }

        int bestCost = std::numeric_limits<int>::max();
        Position bestStairOrigin, bestStairDest;
        bool bestIsActive = false;

        for (const auto& trans : stairs) {
            if (trans.origin.z != getPosition().z) continue;
            auto path = g_globalMonsterMap().findPath(getPosition(), trans.origin);
            if (path.empty()) continue;
            int distToStair = static_cast<int>(path.size());
            int distDestToPlayer = std::abs(trans.destination.x - leader->getPosition().x)
                                 + std::abs(trans.destination.y - leader->getPosition().y);
            int cost = distToStair + distDestToPlayer;
            if (cost < bestCost) {
                bestCost = cost;
                bestStairOrigin = trans.origin;
                bestStairDest = trans.destination;
                bestIsActive = trans.isActive;
            }
        }

        if (bestCost < std::numeric_limits<int>::max()) {
            m_stairOrigin = bestStairOrigin;
            m_stairDestination = bestStairDest;
            m_stairIsActive = bestIsActive;
            m_pursuingStairs = true;
            m_stairStuckCount = 0;
            m_stairTotalCycles = 0;
            m_stairPathfindFailCount = 0;
            m_forcedMoveFailCount = 0;
            m_proactiveFailCount = 0;
            m_savedFollowId = leader->getID();
            setFollowCreature(nullptr);
            g_logger().info("[MCR-COGNI] {} perseguição proactiva: escada em {} (custo={})",
                getName(), bestStairOrigin.toString(), bestCost);
        } else {
            m_proactiveFailCount++;
        }
    } else {
        m_proactiveFailCount++;
    }
}

void Monster::tryExploreArea() {
    uint64_t now = getCurrentTimeMs();
    if (now - m_lastExplorationScan < 30000) return;
    m_lastExplorationScan = now;

    Position myPos = getPosition();

    // Verificar se já temos cobertura suficiente ao redor
    int coveredTiles = 0;
    for (int dx = -20; dx <= 20; ++dx) {
        for (int dy = -20; dy <= 20; ++dy) {
            if (dx == 0 && dy == 0) continue;
            Position check(myPos.x + dx, myPos.y + dy, myPos.z);
            if (g_globalMonsterMap().isWalkable(check)) coveredTiles++;
        }
    }
    // Se já temos >60% de cobertura num raio de 15, não vale a pena escanear
    if (coveredTiles > 540) return;

    g_logger().info("[MCR-EXPLORE] {} scan exploratório concluído. Tiles no mapa: {}",
        getName(), g_globalMonsterMap().getTiles().size());

    g_globalMonsterMap().scanArea(myPos, 30, 2, 2, 1500);
}

void Monster::tryAlternativeRouteToLeader() {
    if (m_usingGlobalPath || m_pursuingStairs || m_returningToSpawn || m_returningHome) return;

    auto target = getAttackedCreature();
    if (!target) target = getFollowCreature();
    if (!target || !target->getPlayer()) return;

    Position myPos = getPosition();
    Position targetPos = target->getPosition();

    if (myPos.z != targetPos.z) return;

    FindPathParams fpp;
    getPathSearchParams(getMonster(), fpp);
    std::vector<Direction> dummy;
    if (getPathTo(targetPos, dummy, fpp)) return;

    uint64_t now = getCurrentTimeMs();
    if (now - m_lastGlobalScanAttempt < 3000) return;
    m_lastGlobalScanAttempt = now;

    if (!g_globalMonsterMap().isWalkable(myPos)) {
        size_t dummyAdded;
        mcrTryScanArea(myPos, 20, 1, 1, 500,
            target->getPlayer()->getStairTransitions(), &dummyAdded);
    }

    auto fullPath = g_globalMonsterMap().findPath(myPos, targetPos);
    if (!fullPath.empty()) {
        clearGlobalPath();
        m_globalPath = fullPath;
        m_globalPathIndex = 0;
        m_usingGlobalPath = true;
        m_savedFollowId = target->getID();
        setFollowCreature(nullptr);
        g_logger().info("[MCR-ROUTE] {} rota alternativa encontrada: {} passos",
            getName(), fullPath.size());
        return;
    }

    auto nearestPath = g_globalMonsterMap().findNearestApproachTo(myPos, targetPos);
    if (!nearestPath.empty()) {
        clearGlobalPath();
        m_globalPath = nearestPath;
        m_globalPathIndex = 0;
        m_usingGlobalPath = true;
        m_savedFollowId = target->getID();
        setFollowCreature(nullptr);
        g_logger().info("[MCR-ROUTE] {} aproximação máxima: {} passos",
            getName(), nearestPath.size());
    }
}
Monster::followGlobalPathStep()

void Monster::followGlobalPathStep() {
    if (!m_usingGlobalPath || m_globalPathIndex >= m_globalPath.size()) {
        clearGlobalPath();
        return;
    }

    Position myPos = getPosition();
    Position nextPos = m_globalPath[m_globalPathIndex];

    if (myPos == nextPos) {
        m_globalPathIndex++;
        if (m_globalPathIndex >= m_globalPath.size()) { clearGlobalPath(); return; }
        nextPos = m_globalPath[m_globalPathIndex];
    }

    if (m_pursuingStairs && (myPos == m_stairOrigin ||
        std::abs(myPos.x - m_stairOrigin.x) + std::abs(myPos.y - m_stairOrigin.y) <= 1)) {
        clearGlobalPath();
        return;
    }

    Direction dir = getDirectionTo(myPos, nextPos);
    if (dir == DIRECTION_NONE) {
        m_globalPathIndex++;
        if (m_globalPathIndex >= m_globalPath.size()) clearGlobalPath();
        return;
    }

    Position oldPos = myPos;
    ReturnValue ret = g_game().internalMoveCreature(static_self_cast<Monster>(), dir, 0);
    if (ret != RETURNVALUE_NOERROR)
        ret = g_game().internalMoveCreature(static_self_cast<Monster>(), dir, FLAG_IGNOREBLOCKITEM);

    if (getPosition() == oldPos) {
        m_globalPathIndex++;
        if (m_globalPathIndex >= m_globalPath.size()) { clearGlobalPath(); return; }
        nextPos = m_globalPath[m_globalPathIndex];
        dir = getDirectionTo(myPos, nextPos);
        if (dir != DIRECTION_NONE) {
            ret = g_game().internalMoveCreature(static_self_cast<Monster>(), dir, 0);
            if (ret != RETURNVALUE_NOERROR)
                ret = g_game().internalMoveCreature(static_self_cast<Monster>(), dir, FLAG_IGNOREBLOCKITEM);
        }
        if (getPosition() == oldPos) clearGlobalPath();
    }
}
Monster::computePursuitDeadline()

uint64_t Monster::computePursuitDeadline() {
    int baseTimeMs = 25000;
    int engagementTimeMs = m_engagement * 350;
    int totalMs = baseTimeMs + engagementTimeMs;
    if (totalMs > 45000) totalMs = 45000;
    return getCurrentTimeMs() + totalMs;
}

void Monster::invalidateTileBlocked(const Position& pos) {
    auto tile = g_game().map.getTile(pos);
    if (!tile) return;
    if (tile->getTopCreature()) return;

    bool blocked = tile->hasFlag(TILESTATE_BLOCKPATH) || tile->hasFlag(TILESTATE_BLOCKSOLID);
    if (!blocked) return;

    if (tile->hasFlag(TILESTATE_BLOCKSOLID)) {
        g_globalMonsterMap().addTile(pos, false, false);
        return;
    }

    g_globalMonsterMap().addTemporaryObstacle(pos, 15000);
}

Position Monster::findFreeTileNear(const Position& center, int maxRadius) const {
    for (int r = 1; r <= maxRadius; ++r) {
        for (int dx = -r; dx <= r; ++dx) {
            for (int dy = -r; dy <= r; ++dy) {
                if (std::abs(dx) != r && std::abs(dy) != r) continue;
                Position check(center.x + dx, center.y + dy, center.z);
                auto tile = g_game().map.getTile(check);
                if (!tile) continue;
                if (tile->hasFlag(TILESTATE_BLOCKPATH) || tile->hasFlag(TILESTATE_BLOCKSOLID)) continue;
                auto topCreature = tile->getTopCreature();
                if (topCreature && topCreature->getPlayer()) continue;
                if (!topCreature) return check;
            }
        }
    }
    return Position();
}

void Monster::mcrSafeTeleport(const Position& target) {
    Position safePos = findFreeTileNear(target, 4);
    if (safePos.x != 0 || safePos.y != 0) {
        g_game().internalTeleport(static_self_cast<Monster>(), safePos);
        g_game().addMagicEffect(safePos, CONST_ME_TELEPORT);
        return;
    }
    auto centerTile = g_game().map.getTile(target);
    if (centerTile && !centerTile->hasFlag(TILESTATE_BLOCKPATH) && !centerTile->getTopCreature()) {
        g_game().internalTeleport(static_self_cast<Monster>(), target);
    }
}

void Monster::handleSummonMasterTeleport(Player* player) {
    mcrSafeTeleport(player->getPosition());
}
