# Licoes.md — Lições Aprendidas MCR

> Parte do sistema modular `docs/rules/`. Consulte `../AGENTS.md` para visão geral.

## Propósito

Registrar aprendizados importantes durante o desenvolvimento para não repetir erros.

## O Sistema

- Lições são gerenciadas por `scripts/lesson.py`
- As mais recentes ficam em `docs/lessons/recentes.md`
- Consulte `Scripts/README_AUTONOMY.md` para detalhes

## Regras

1. **Salve uma lição** sempre que:
   - Resolver um bug complexo
   - Descobrir algo relevante não documentado
   - Tomar decisão arquitetural importante

2. **Ao iniciar conversa**, após Pendências.md e checkpoint, revise as lições recentes.

3. **Compartilhe lições com o Hub do Lojista** se aplicáveis ao contexto de varejo.
