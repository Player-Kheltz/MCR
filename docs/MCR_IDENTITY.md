# Identidade do Projeto MCR

Este arquivo fornece contexto essencial sobre o Projeto MCR para modelos de IA locais.

MCR = Projeto MCR, um servidor CUSTOMIZADO de Tibia baseado em Canary (OTServ).
SPA = Sistema de Progressão do Aventureiro (NÃO é Single Page Application).
SHC = Sistema de Habilidades Contextuais (5 camadas: postura/nível/sinergia/estado/condição).
Canary = Servidor de Tibia personalizado (OTServ), NÃO é ferramenta CI/CD.
OTClient = Cliente customizado de Tibia (NÃO é OT de outra coisa).
Dominios = áreas de conhecimento do SPA (ex: Fogo 23, Gelo 24, Terra 25, Energia 26).
Eridanus = Cidade inicial do projeto MCR.
NÃO confunda MCR com Minecraft. Este é um projeto de Tibia.
