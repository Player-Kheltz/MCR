# PLAN - Arquitetura do Dominio CRISTAL
# Modelo: deepseek-r1:8b
# Tempo: 15.727262258529663s

**Dominio CRISTAL (ID 27)**

1. **Essencia**:
   - Adjetivos: Imaculado, Concentrado, Pristína.
   - Frase de assinatura: *"Refino da mente, poder da forma."*

2. **Comportamento de Combate**:
   - **Força**: Capaz de manipular poderes elementais com precisão e eficiência, criando armas e defesas temporárias.
   - **Fraqueza**: Poder concentrado, frágil e suscetível a distrações ou ataques psicológicos.

3. **4 Pacotes Térmicos de Habilidades**:
   - **Manipulação Direta**:
     - Cristalizar: Criar cristais de energia para ataques diretos.
     - Polimento: Aumentar a precisão de ataques em 10%.
   - **Armazenamento**:
     - Reserva de Cristal: Armazenar energia para uso futuro.
     - Concentração: Recuperar magia perdida.
   - **Transformação**:
     - Fusão Elemental: Combinar elementos para criar novas formas.
     - Refinação: Transformar habilidades em cristais.
   - **Defesa**:
     - Blindagem Cristalina: Criar barreiras temporárias.
     - Fragilidade: Reduzir danos, mas torna-se frágil.

4. **Sinergias com outros dominios**:
   - **Fogo**: Combina com Fogo 23 para criar chamas cristalizadas.
   - **Gelo**: Permite a fragmentação de cristais para ataques de frio.
   - **Terra**: Cria cristais geológicos para fortalecer o terreno.
   - **Energia**: Amplifica a energia vital, criando cristais de poder.

5. **Identidade Visual**:
   - Efeitos: CONST_ME_CRYSTAL, CONST_ME_GLOW, CONST_ME_LIGHT.

6. **Nicho Principal**:
   - O dominio CRISTAL permite aos jogadores manipular elementos de forma eficiente e transformá-los em poderes únicos, oferecendo uma experiência de combate inteligente e estratégica.
