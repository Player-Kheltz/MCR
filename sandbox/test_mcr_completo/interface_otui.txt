Aqui está um exemplo de layout OTUI para uma janela de "Status do Personagem" no OTClient:

```otui
<Window id="CharacterStatusWindow" title="Status do Personagem" width="300" height="250">
    <Anchor x="10" y="10">
        <Label text="Nome:" font="Arial" size="14" color="#FFFFFF"/>
        <Label id="NameLabel" text="" font="Arial" size="14" color="#FFFFFF"/>
    </Anchor>
    
    <Anchor x="10" y="40">
        <Label text="Nível:" font="Arial" size="14" color="#FFFFFF"/>
        <Label id="LevelLabel" text="" font="Arial" size="14" color="#FFFFFF"/>
    </Anchor>
    
    <Anchor x="10" y="70">
        <Label text="Vocação:" font="Arial" size="14" color="#FFFFFF"/>
        <Label id="VocationLabel" text="" font="Arial" size="14" color="#FFFFFF"/>
    </Anchor>
    
    <Anchor x="10" y="100">
        <Label text="Saúde:" font="Arial" size="14" color="#FFFFFF"/>
        <ProgressBar id="HealthBar" width="280" height="20" min="0" max="1000" value="500" color="#FF0000"/>
    </Anchor>
    
    <Anchor x="10" y="130">
        <Label text="Mana:" font="Arial" size="14" color="#FFFFFF"/>
        <ProgressBar id="ManaBar" width="280" height="20" min="0" max="500" value="250" color="#0000FF"/>
    </Anchor>
    
    <Anchor x="10" y="160">
        <Label text="Capacidade:" font="Arial" size="14" color="#FFFFFF"/>
        <ProgressBar id="CapacityBar" width="280" height="20" min="0" max="100" value="50" color="#00FF00"/>
    </Anchor>
</Window>
```

### Explicação dos Elementos:

- **Window**: Define a janela principal com um título e dimensões.
- **Anchor**: Posiciona os elementos dentro da janela. O `x` e `y` definem as coordenadas do ânculo de posicionamento.
- **Label**: Exibe texto estático ou dinâmico (usando IDs para atualização).
- **ProgressBar**: Exibe uma barra de progresso com valor, cor e dimensões.

### Como Atualizar os Valores:

Para atualizar os valores dos rótulos e barras de progresso, você precisará usar um script Lua ou outra linguagem de programação compatível com o OTClient. Aqui está um exemplo simples em Lua para atualizar os valores:

```lua
local window = getInterface("CharacterStatusWindow")

if window then
    window.NameLabel.text = "Seu Nome"
    window.LevelLabel.text = "100"
    window.VocationLabel.text = "Mago"
    
    window.HealthBar.value = 750
    window.ManaBar.value = 300
    window.CapacityBar.value = 80
end
```

Este script deve ser executado em um contexto onde você tem acesso ao OTClient e pode interagir com a interface gráfica.
