## Comando cmd_listar_kg.py - Codigo Completo

Segue o comando seguindo o padrao MCR-DevIA (register + execute):

```python
"""Comando: listar_kg - Lista estatisticas do Knowledge Graph."""
import os
from collections import Counter

def register():
    return {
        "name": "listar_kg",
        "desc": "Lista estatisticas do KG: total, ativas/inativas, top categorias, recentes. Uso: listar_kg",
        "handler": execute,
        "args": [],
        "categoria": "consulta",
    }

def execute(kg, ia, args, ctx_crew=None):
    """Lista estatisticas do Knowledge Graph."""
    if not kg:
        print("[KG] Knowledge Graph nao disponivel")
        return True

    # (a) Total de registros
    todas = kg.listar() if hasattr(kg, 'listar') else []
    total = len(todas)

    # (b) Ativas vs Inativas
    ativas = [l for l in todas if not l.get('inactive', False)]
    inativas = [l for l in todas if l.get('inactive', False)]

    # (c) Top 5 categorias
    categorias = Counter(l.get('ctx', 'geral') for l in todas)
    top5 = categorias.most_common(5)

    # (d) 3 mais recentes
    recentes = sorted(todas, key=lambda l: l.get('ts', 0), reverse=True)[:3]

    # Relatorio
    print("=" * 50)
    print("  KNOWLEDGE GRAPH - ESTATISTICAS")
    print("=" * 50)
    print(f"  Total de lessons: {total}")
    print(f"  Ativas: {len(ativas)}")
    print(f"  Inativas: {len(inativas)}")
    print()
    print("  Top 5 Categorias:")
    for cat, qtd in top5:
        print(f"    {cat}: {qtd}")
    print()
    print("  3 Lessons mais recentes:")
    for l in recentes:
        erro = l.get('erro', 'sem titulo')[:60]
        print(f"    [{l.get('ctx', '?')}] {erro}")

    return True
```

### Observacoes sobre o design:
- Usei `hasattr(kg, 'listar')` para ser防御ivo - se o KG nao tiver metodo listar, retorna lista vazia
- O `Counter` e importando dentro do escopo correto (fora da funcao para eficiencia)
- As chaves 'inactive', 'ctx', 'ts', 'erro' seguem o padrao observado em outros comandos do projeto
- Nao usei `ctx_crew` porque este comando e puramente consulta ao KG
- A saida e em formato tabular legivel, consistente com cmd_status.py
