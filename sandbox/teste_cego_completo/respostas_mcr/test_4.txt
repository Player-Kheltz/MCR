Entendido! Vou criar um comando Python chamado `cmd_listar_kg.py` seguindo o padrão do projeto existente, conforme solicitado. O comando irá listar informações sobre o KG: total de registros, ativos/inativos, top 5 categorias e os 3 registros mais recentes.

```python
# cmd_listar_kg.py

def register():
    return {
        "name": "listar_kg",
        "desc": "Lista informações sobre o KG: total de registros, ativos/inativos, top 5 categorias e os 3 mais recentes.",
        "handler": execute,
        "args": {}
    }

def execute(kg, ia, args, ctx_crew=None):
    # Total de registros
    total_registros = len(kg)

    # Ativos vs Inativos
    ativos = sum(1 for item in kg if item['status'] == 'ativo')
    inativos = total_registros - ativos

    # Top 5 categorias
    categorias = {}
    for item in kg:
        categoria = item.get('categoria', 'sem_categoria')
        if categoria in categorias:
            categorias[categoria] += 1
        else:
            categorias[categoria] = 1
    
    top_5_categorias = sorted(categorias.items(), key=lambda x: x[1], reverse=True)[:5]

    # Os 3 registros mais recentes
    registros_recentes = sorted(kg, key=lambda x: x['data'], reverse=True)[:3]

    # Montando a resposta
    resposta = f"Total de registros: {total_registros}\n"
    resposta += f"Ativos: {ativos}, Inativos: {inativos}\n"
    resposta += "Top 5 categorias:\n"
    for categoria, quantidade in top_5_categorias:
        resposta += f"- {categoria}: {quantidade} registros\n"
    resposta += "Os 3 registros mais recentes:\n"
    for registro in registros_recentes:
        resposta += f"- {registro['nome']}, Data: {registro['data']}\n"

    return resposta
```

Este código implementa o comando `listar_kg` conforme solicitado. Ele utiliza as informações armazenadas no KG (Knowledge Graph) para calcular e listar os dados desejados. Certifique-se de que o formato dos registros em seu KG seja compatível com a estrutura esperada neste script.
