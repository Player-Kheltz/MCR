# Sandbox do OC-Dev

Area de testes para o assistente de desenvolvimento local (Qwen 2.5 Coder 7B).

## Regras
- Voce pode criar, editar e excluir QUALQUER arquivo dentro de `sandbox/`
- Para arquivos fora daqui, voce deve pedir permissao
- Ferramentas de leitura (read, grep, glob) funcionam em todo o projeto

## Estrutura
- `testes/` — Scripts para criar e modificar
- `bugs/` — Codigo com erros para corrigir
- `README.md` — Este arquivo
