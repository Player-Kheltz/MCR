A melhor abordagem para limitar a visão ao apenas 1 piso (sem entre-pisos) é modificar `m_mapUndergroundFloorRange` de 2 para 0 ou 1 no arquivo `gameconfig.h`. Isso garante que o jogo não permita a visualização de pisos subterrâneos.

### Modificação em gameconfig.h

**Linha a ser modificada:**
```cpp
uint8_t getMapUndergroundFloorRange() const { return m_mapUndergroundFloorRange; }
```

**Conteúdo antes da alteração:**
```cpp
uint8_t m_mapUndergroundFloorRange = 2;
```

**Conteúdo depois da alteração:**
```cpp
uint8_t m_mapUndergroundFloorRange = 0; // Ou 1, dependendo do comportamento desejado
```

### Arquivo de Configuração Lua

Para permitir ativar/desativar esse comportamento através de um arquivo de configuração Lua, você pode adicionar uma opção no `config.lua` ou em outro arquivo de configuração.

**Exemplo de config.lua:**
```lua
-- config.lua
mapUndergroundFloorRange = 0 -- Ou 1, dependendo do comportamento desejado
```

### Implementação no MapView

Para aplicar a configuração lida do `config.lua`, você pode modificar o construtor ou um método de inicialização do `MapView` para ler essa configuração e ajustar `m_mapUndergroundFloorRange`.

**Exemplo de modificação em mapview.cpp:**

**Linha a ser modificada:**
```cpp
m_floors.resize(g_gameConfig.getMapMaxZ() + 1);
```

**Conteúdo antes da alteração:**
```cpp
m_floors.resize(g_gameConfig.getMapMaxZ() + 1);
```

**Conteúdo depois da alteração:**
```cpp
m_mapUndergroundFloorRange = g_config.getNumber("mapUndergroundFloorRange", 2); // Ler a configuração do arquivo Lua
m_floors.resize(g_gameConfig.getMapMaxZ() + m_mapUndergroundFloorRange);
```

### Resumo das Modificações

1. **gameconfig.h:**
   - Alterar `uint8_t m_mapUndergroundFloorRange = 2;` para `uint8_t m_mapUndergroundFloorRange = 0;` ou `uint8_t m_mapUndergroundFloorRange = 1;`

2. **mapview.cpp:**
   - Adicionar a leitura da configuração do arquivo Lua:
     ```cpp
     m_mapUndergroundFloorRange = g_config.getNumber("mapUndergroundFloorRange", 2);
     ```
   - Modificar o tamanho dos vetores de pisos:
     ```cpp
     m_floors.resize(g_gameConfig.getMapMaxZ() + m_mapUndergroundFloorRange);
     ```

3. **config.lua:**
   - Adicionar a opção para controlar o comportamento:
     ```lua
     mapUndergroundFloorRange = 0 -- Ou 1, dependendo do comportamento desejado
     ```

Essas modificações garantirão que o jogo visualize apenas um piso (sem entre-pisos) conforme configurado.
