PROMPT: Write a Python function to check if a number is prime. Include comments.

RESPOSTA:
To determine if a given number is prime, we can follow a systematic approach:

1. **Handle Edge Cases**: Numbers less than 2 are not prime. If the number is exactly 2, it's prime. Even numbers greater than 2 are not prime.

2. **Check Divisors Efficiently**: For numbers greater than 2, check divisibility by odd numbers up to the square root of the given number. This reduces the number of checks needed compared to checking all possible divisors up to the number itself.

Here is a Python function that implements this logic:

```python
import math

def is_prime(n):
    if n < 2:
        return False
    elif n == 2:
        return True
    elif n % 2 == 0:
        return False
    sqrt_n = math.isqrt(n)
    for i in range(3, sqrt_n + 1, 2):
        if n % i == 0:
            return False
    return True
```

**Explanation of the Function:**

- **Edge Cases Handling**: The function immediately returns `False` for numbers less than 2. For even numbers greater than 2, it also returns `False`.
  
- **Square Root Optimization**: By checking divisors only up to the square root of `n`, we significantly reduce the number of iterations needed.

- **Efficient Looping**: The loop starts from 3 and checks every odd number up to the integer square root of `n`. This ensures that all potential factors are checked efficiently without redundant even numbers.

This function correctly identifies prime numbers by leveraging mathematical properties and efficient looping, ensuring both correctness and performance.
