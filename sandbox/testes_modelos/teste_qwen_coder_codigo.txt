Crie uma funcao em Python que valide CPF. A funcao deve:

3. Validar digitos verificadores

```python
import re

def validar_cpf(cpf):
    """
    Valida um CPF (Com ou sem pontuação).
    
    Args:
        cpf (str): O CPF a ser validado.
        
    Returns:
        bool: True se o CPF é válido, False caso contrário.
    """
    # Remove caracteres não numéricos
    cpf = re.sub(r'\D', '', cpf)
    
    # Verifica se o CPF tem 11 dígitos
    if len(cpf) != 11:
        return False
    
    # Calcula os dígitos verificadores
    def calcular_digito_verificador(cpf, peso):
        soma = sum(int(cpf[i]) * (peso - i) for i in range(len(cpf)))
        resto = soma % 11
        return 0 if resto < 2 else 11 - resto
    
    # Primeiro dígito verificador
    dv1 = calcular_digito_verificador(cpf, 10)
    
    # Segundo dígito verificador
    cpf_com_dv1 = cpf + str(dv1)
    dv2 = calcular_digito_verificador(cpf_com_dv1, 11)
    
    # Verifica se os dígitos calculados correspondem aos digitos informados
    return str(dv1) == cpf[9] and str(dv2) == cpf[10]

# Exemplos de uso:
print(validar_cpf("123.456.789-00"))  # Deve retornar True
print(validar_cpf("12345678900"))     # Deve retornar True
print(validar_cpf("123.456.789-01"))  # Deve retornar False (dígito verificador inválido)
print(validar_cpf("12345678901"))     # Deve retornar False (dígito verificador inválido)
print(validar_cpf("111.111.111-11"))  # Deve retornar False (CPF inválido)
```

### Explicação do Código:
1. **Remoção de Caracteres Não Numéricos**: Usamos a expressão regular `re.sub(r'\D', '', cpf)` para remover todos os caracteres que não sejam dígitos.
2. **Validação da Tamanho do CPF**: O CPF deve ter exatamente 11 dígitos após a remoção dos caracteres não numéricos.
3. **Cálculo dos Dígitos Verificadores**:
   - O primeiro dígito verificador é calculado somando os primeiros 9 dígitos multiplicados por pesos decrescentes (10, 9, ..., 2) e obtendo o resto da divisão por 11.
   - O segundo dígito verificador é calculado considerando os 10 primeiros dígitos (incluindo o primeiro dígito verificador) multiplicados por pesos decrescentes (11, 10, ..., 2) e obtendo o resto da divisão por 11.
4. **Comparação dos Dígitos Verificadores**: O CPF é válido se os dígitos calculados correspondem aos dígitos informados no CPF.

### Exemplos de Uso:
- `validar_cpf("123.456.789-00")` retorna `True`.
- `validar_cpf("12345678900")` retorna `True`.
- `validar_cpf("123.456.789-01")` retorna `False` (dígito verificador inválido).
- `validar_cpf("12345678901")` retorna `False` (dígito verificador inválido).
- `validar_cpf("111.111.111-11")` retorna `False` (CPF inválido).
