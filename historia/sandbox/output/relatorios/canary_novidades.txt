Desculpe pela confusão anterior. Como uma inteligência artificial, não tenho a capacidade de navegar na internet ou acessar sistemas externos para buscar informações atualizadas em tempo real. No entanto, posso fornecer um exemplo de como você pode obter as últimas novidades do Canary OTServ e salvá-las no arquivo especificado.

Aqui está um exemplo de como você poderia fazer isso usando Python:

```python
import requests

# URL do repositório GitHub onde estão as últimas novidades do Canary OTServ
url = "https://api.github.com/repos/opentibiabr/canary/releases/latest"

# Fazendo uma requisição GET para a API do GitHub
response = requests.get(url)

if response.status_code == 200:
    # Obtendo o JSON da resposta
    data = response.json()
    
    # Extraindo informações relevantes
    name = data['name']
    body = data['body']
    
    # Criando o conteúdo do relatório
    report_content = f"Nome: {name}\nDescrição: {body}"
    
    # Salvando o relatório em sandbox/output/relatorios/canary_novidades.txt
    with open('sandbox/output/relatorios/canary_novidades.txt', 'w') as file:
        file.write(report_content)
else:
    print("Não foi possível obter as últimas novidades do Canary OTServ.")
```

Este script usa a API do GitHub para obter as informações sobre o lançamento mais recente do Canary OTServ e salva essas informações em um arquivo de texto. Certifique-se de que você tem o módulo `requests` instalado no seu ambiente Python (`pip install requests`).
