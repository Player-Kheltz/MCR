=====================================================================
1. INTRODUÇÃO
=====================================================================
A Matriz de Sinergias define interações entre domínios no SPA.
Cada sinergia é ativada quando o jogador possui nível mínimo em
um domínio secundário, adicionando efeitos extras às habilidades
do domínio primário.

As sinergias são definidas no bloco `sinergias` de cada habilidade
e processadas pelo Resolvedor Contextual (contexto.lua).

=====================================================================
2. MATRIZ COMPLETA
=====================================================================

LEGENDA:
  • D → D: Domínio de Origem → Domínio Alvo
  • Nv: Nível mínimo requerido no domínio alvo
  • Efeito: O que a sinergia adiciona

=====================================================================
3. HIERARQUIA DE PROPAGAÇÃO (Rácio 4:2:1)
=====================================================================

Quando o jogador ganha afinidade numa Especialidade:
  • 100% dos pontos para a Especialidade
  • 50% para o Domínio Secundário (pai)
  • 25% para o Domínio Primário (avô)

Exemplo: Espadas Leves (100) → Lâminas (10) → Combate (1)
  • +10 afinidade em Espadas Leves
  • +5 afinidade em Lâminas
  • +2.5 afinidade em Combate

=====================================================================
4. NOTAS DE IMPLEMENTAÇÃO
=====================================================================

1. Sinergias são definidas dentro de cada HABILIDADES[ID].sinergias{}
2. O Resolvedor Contextual (contexto.lua) verifica sinergias ativas
3. nivelMin = nível do domínio alvo necessário para ativar
4. Múltiplas sinergias podem stackar no mesmo efeitoConfig
5. Sinergias de nível 10+ são reservadas para efeitos poderosos (ultimates)
6. A Síntese Superior (Primário nível 20) é definida em progressao.lua
