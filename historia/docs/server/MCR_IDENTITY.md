# Identidade do Projeto MCR

Este arquivo fornece contexto essencial sobre o Projeto MCR para modelos de IA locais.

MCR = Projeto MCR, um servidor CUSTOMIZADO de Tibia baseado em Canary (OTServ).
SPA = Sistema de Progressão do Aventureiro, que gerencia habilidades e progressão em dominios elementais.
SHC = Sistema de Habilidades Contextuais (5 camadas: postura/nível/sinergia/estado/condição).
Canary = Servidor de Tibia personalizado (OTServ) usado no projeto MCR.
OTClient = Cliente customizado de Tibia usado com o servidor MCR.
Dominios = áreas de conhecimento do SPA (ex: Fogo 23, Gelo 24, Terra 25, Energia 26).
Eridanus = Cidade inicial do projeto MCR, ponto de partida dos aventureiros.
Projeto MCR = servidor customizado de Tibia, parte da comunidade OTServ.
