# 2026-06-23 — Canal Assistente (500)

## Decisao
Criar canal de chat "Assistente" via chatchannels.xml + script Lua com onSpeak, em vez de modificar o C++ ou usar apenas TalkAction.

## Motivo
- Canary ja suporta canais com eventos Lua (onSpeak, canJoin)
- Nao precisa recompilar o servidor
- OTClient abre aba automaticamente via joinChannel
- Mensagens sao roteadas pelo protocolo nativo (opcode 0x96)
- Fallback !assistente mantido para compatibilidade

## Alternativas rejeitadas
- Modificar C++ Game::playerSay: risco alto, precisa recompilar
- Apenas TalkAction: nao teria aba separada no chat do cliente

## Referencias
- `data/chatchannels/chatchannels.xml` — registro do canal 500
- `data/chatchannels/scripts/assistente.lua` — script do canal
- `OTClient/modules/game_console/console.lua` — aba no cliente
- `scripts/bridge_auto.py` — tratamento do prefixo CHANNEL|
